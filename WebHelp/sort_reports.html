Sort Reports

You can generate a number of reports using MPC. These reports can be generated based on the filter criteria. Reports can be scheduled, previewed, and printed.

To sort reports:

  1. In the MPC Dashboard area, click the REPORTS tab. The REPORTS page appears. A list of all the reports, with the schedule and status, appears.
  2. Under Card Reports column, click the required report, which appears as a hyperlink.
  3. Perform the following functions to set filters for the report.
    4. Field Description
    Sort
    Sort Order1 Set the first type of order in which the report must be filtered.
    Sort Order2 Set the second type of order in which the report must be filtered.
    Sort Order 3 Set the third type of order in which the report must be filtered.
  4. Click Preview.
