Create Reports

The Create Reports option enables you to create the following reports:

  • Audit
  • Card
  • Event

To create a report:

  1. In the MPC Dashboard area, click the Reports tab.
  2. Click Create Reports. The REPORTS page appears.
  3. Under Reports column, select the required report. For example, select CARD REPORTS
  4. Perform the following functions.
Field Description
