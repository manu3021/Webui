Firmware-Aktualisierung

Zum Bearbeiten der Geräteservices:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Im linken Fensterbereich den Standort auswählen.
  3. Auf den Reiter Firmware aktualisieren klicken. Eine Liste mit allen registrierten Geräten wird angezeigt.
  4. Aus der Dropdown-Liste Gerätemodell auswählen das Gerät auswählen. Es werden alle für das ausgewählte Gerät verfügbaren Firmware angezeigt.
  5. Aus der Dropdown-Liste Aktualisieren auf die entsprechende Version für das ausgewählte Gerät auswählen.
  6. Aus der Liste Gerätename das einzelne Gerät auswählen.
  7. In der Liste Status aktualisieren auf Aktualisieren klicken.
  8. Alle ausgewählten Firmware werden auf die gewünschte Version aktualisiert.

Hinweis:

Die Option Firmware aktualisieren ermöglicht Ihnen die Auswahl einzelner Geräte zur Aktualisierung. Ein Bulk-Upgrade (Massenaktualisierung) ist bei Firmware-Aktualisierung nicht möglich. Sie müssen zum Beispiel jeweils die Firmware für ein Gerät auswählen und anschließend auf Aktualisieren klicken.

Während der Firmware-Aktualisierung, falls Sie ein HRRH8-Gerät ausgewählt haben, empfiehlt Honeywell die Auswahl einer Firmwareversion, die der aktuellen nachfolgt.
