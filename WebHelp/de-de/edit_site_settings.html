Standorteinstellungen bearbeiten

Zum Bearbeiten der Standortservices:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Im linken Fensterbereich den Standort auswählen. Die Seite Einstellungen wird angezeigt.
  3. Auf das gepunktete Layout klicken, um den Grundriss des ausgewählten Standorts hochzuladen.

    4. Hinweis:

    Es können nur Grundrisse im .JPG- oder .PNG-Format hochgeladen werden.

  4. Die erforderlichen Informationen eingeben und auf Speichern klicken.
