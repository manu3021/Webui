Profil verwalten

Sie können Händlerinformationen, Ihre Profileinstellungen usw., die in Profil gespeichert sind, bearbeiten.

Zum Bearbeiten eines Profils:

  1. Im Reiter Profil auf Mein Profil klicken. Die Seite PROFIL wird angezeigt.
  2. Im Reiter Händlerinformationen müssen Sie:
    • Die Kontoinformationen des Händlers eingeben.
    • Das Profilbild des Händlers hochladen.
    • Das entsprechende autorisierte Händlerprogramm auswählen.
  3. Auf Weiter klicken.
    Oder
    Auf den Reiter Mein Profil klicken.
  4. Im Reiter Mein Profil müssen Sie:
    • Die Kontoinformationen des Kunden eingeben.
    • Das Profilbild des Kunden hochladen.
  5. Auf Speichern klicken.
    Das Profil wurde geändert.

Hinweis:

Zum Ändern des alten Passworts:

  1. Passwort ändern klicken.
  2. Altes Passwort eingeben.
  3. Neues Passwort eingeben.
  4. Passwort ändern klicken.
