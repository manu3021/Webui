Geräteeinstellungen bearbeiten

Zum Bearbeiten der HRRH8-Einstellungen:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Im linken Fensterbereich den Standort auswählen, anschließend das entsprechende Gerät, das dem Standort zugeordnet ist, doppelklicken. Die Seite Einstellungen wird angezeigt.

    3. Hinweis:

    Bevor Sie die Geräteeinstellungen bearbeiten, stellen Sie sicher, dass Sie die Geräte im Cloud registrieren.

  3. Die erforderlichen Daten in folgenden Feldern eingeben:
    • Unter NETZWERKKONFIGURATION,
      • Eindeutiger Schlüssel: Den MACID des Geräts eingeben. Die MACID muss dem Panel entnommen werden.
      • IP-Adresse
      • Subnetzmaske
      • DNS-Server
      • HTTP-Port
      • Gateway
    • Unter EINSTELLUNGEN FÜR AUFZEICHNUNG,
      • Name des Rekorders
      • Hardware-Version
      • Videoformat
      • MAC-Adresse
      • Version der Firmware
    • Unter MAXPRO VERBINDUNGSEINSTELLUNGEN,
      • Aktivieren: Zum Ändern der Server-URL.
    • Unter ERWEITERTE EINSTELLUNGEN,
      • Passwort wiederherstellen
      • Passwort aktualisieren
    • Zum Konfigurieren der HRRH8-Kameraeinstellungen siehe Zum Konfigurieren des Kameras unter HRRH8

Zum Bearbeiten der NetAXS-123-Einstellungen:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Im linken Fensterbereich den Standort auswählen, anschließend das entsprechende Gerät, das dem Standort zugeordnet ist, doppelklicken. Die Seite Einstellungen wird angezeigt.

    3. Hinweis:

    Bevor Sie die Geräteeinstellungen bearbeiten, stellen Sie sicher, dass Sie die Geräte im Cloud registrieren.

  3. Die erforderlichen Daten in folgenden Feldern eingeben:
    • Name des Panels
    • Beschreibung
    • Kommunikation
    • Paneladresse
    • Firmware
    • E/A-Abrufintervall
    • Schleifenverifizierungsintervall
    • Panel CMD Wiederholungsanzahl
    • Panel CMD Timeout
    • Regionale Zeitzone
    • Datum/Uhrzeit festlegen
    • Anti-Passback: Lokal, Vergessen und Global.
    • Zwang
    • LED umkehren
    • Kontinuierlicher Lesevorgang
    • Freie Ausgangseingaben
  4. Speichern klicken.

  • Zum Konfigurieren der NetAXS-123 Eingabeeinstellungen siehe Zum Konfigurieren der Eingaben für NetAXS-123
  • Zum Konfigurieren der NetAXS-123 Ausgabeeinstellungen siehe Zum Konfigurieren der Ausgaben für NetAXS-123
  • Zum Konfigurieren der NetAXS-123 Lesereinstellungen siehe Zum Konfigurieren des Lesers für NetAXS-123
