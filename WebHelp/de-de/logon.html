Anmelden

Zum Anmelden an der MPC Benutzeroberfläche:

  1. Im Browser die URL der Maxpro Cloud Benutzeroberfläche eingeben. Die Seite MAXPRO Cloud Anmeldung wird angezeigt.
  2. Mit den folgenden Optionen können Sie sich anmelden:
    • Als registrierter Benutzer
    • Als neuer Benutzer

Die folgende Tabelle zeigt die Anmeldeprozedur für neue und registrierte Benutzer.

Sind Sie ein registrierter Benutzer ... Sind Sie ein neuer Benutzer ...
a. Benutzernamen und Passwort eingeben.

Hinweis

Der Benutzername ist die E-Mail-ID, die Sie bei der Registrierung angegeben haben.

b. Auf ANMELDEN klicken.

Hinweis

Stellen Sie vor der Anmeldung sicher, dass Sie die Anmeldeinformationen und die Genehmigung vom Händler erhalten haben.

a. Auf Neues Konto erstellen klicken. Die Seite Händleranmeldung wird angezeigt.
b. Die erforderlichen Informationen eingeben und auf Anmelden klicken.

Nachdem Sie die E-Mail über die Registrierung eines neuen Kontos und deren Aktivierung erhalten haben, fahren Sie mit der Anmeldung fort.

Hinweis:

Falls Sie sich nicht anmelden können, klicken Sie auf Hilfe? um Ihr Passwort zurückzusetzen.
