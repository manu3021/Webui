Gerät erstellen

Zum Erstellen eines Geräts:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Im linken Fensterbereich müssen Sie einen Standort auswählen.
  3. Auf das Symbol Erstellen klicken, anschließend Erstellen>Gerät klickenDer Bildschirm NEUES GERÄT wird angezeigt.

    4. Hinweis:

    Ein Gerät kann nur unter einem Standort erstellt werden.

  4. Gerätetypen aus der Dropdown-Liste auswählen. Es können entweder HRRH8 oder NS123 Panel ausgewählt werden.
  5. Den Eindeutiger Schlüssel (MAC-ID) für das ausgewählte Gerät eingeben.
  6. Speichern klicken.

Die Geräteeinstellungen sind editierbar.

  • Zum Bearbeiten der Geräteeinstellungen auf Geräteeinstellungen bearbeiten klicken.
