Regeln erstellen und aktivieren

Zum Erstellen und Aktivieren von Regeln:

  1. Im Reiter Regeln auf Neue Regel erstellen klicken.
  2. Unter Regelname eine Bezeichnung für die Regel eingeben.
  3. Unter Position den gewünschten Standort eingeben. Durch WENN können Bedingungen für das Gerät, den Zeitplan und die Zeit ausgewählt werden.
  4. Unter Eine Bedingung auswählenkönnen Bedingungen für Folgende eingerichtet werden:
    • Gerät
      • Auf das Symbol im Feld Gerät klicken.
      • Unter GERÄTE AUSWÄHLEN Geräte wie MPCA, NetAXS Panel, Kameras, Tür, Eingang und Ausgang, die hinzugefügt werden sollen, auswählen.
      • Bestätigen klicken.
      • Auf das Symbol im Feld Ereignistyp auswählen klicken.
      • Unter EREIGNISTYP AUSWÄHLEN die Ereignisse auswählen, die eingerichtet werden sollen.
      • Bestätigen klicken.
    • Zeitplan
      • Auf das Symbol im Feld Zeitplan klicken.
      • Unter EREIGNISTYP AUSWÄHLENZeitplan auswählen, der eingerichtet werden soll. Weitere Informationen zur Zeitplankonfiguration siehe Zeitplan erstellen.
      • Bestätigen klicken.
    • Zeitbedingungen
      • Unter Innerhalbdie Pfeiltasten verwenden, um die Min. und Sek. Angaben einzugeben.

    Hinweis:

    Es können auch mehrere UND/ODER Bedingungen eingegeben werden.

  5. Unter DANNauf Aktion auswählen klicken..
    • Wenn Sie Gerätauswählen, dann:
      • Auf das Symbol im Feld Geräte auswählen klicken.
      • Unter GERÄT AUSWÄHLEN Geräte zum Hinzufügen auswählen.
      • Bestätigen klicken.
      • Auf das Symbol im Feld Aktion auswählen klicken.
      • Unter GERÄT AUSWÄHLEN Aktionen, die dem entsprechenden Gerät zugeordnet werden sollen, auswählen.
      • Bestätigen klicken.
    • Wenn Sie Aktionenauswählen, dann:
      • Auf das Symbol im Feld Aktionstyp klicken.
      • Unter AKTIONEN AUSWÄHLENaus Folgenden auswählen:
        • E-MAIL<>UND/ODER
        • PUSH-DIENSTE
      • Bestätigen klicken.
      • Auf das Symbol im Feld Aktionstyp klicken.
        Unter E-MAILS AUSWÄHLENwird eine Liste mit erstellten Operatorennamen angezeigt.
      • Den gewünschten Operatornamen auswählen.
      • Bestätigen klicken.
  6. Regel speichern klicken.

    7. Standardmäßig sind alle neu erstellten Regeln aktiv.
