Zum Konfigurieren des Kameras unter HRRH8

Zum Konfigurieren der Kameraeinstellungen:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Im linken Fensterbereich den Standort auswählen, anschließend das entsprechende Gerät, das dem Standort zugeordnet ist, klicken.

    3. Hinweis:

    Bevor Sie die Geräteeinstellungen bearbeiten, stellen Sie sicher, dass Sie die Geräte im Cloud registrieren.

  3. Im linken Fensterbereich auf das Symbol, das dem HRRH8-Gerät entspricht klicken, um alle verfügbaren Kameras anzuzeigen.
  4. Auf die Kamera doppelklicken. Die Seite Einstellungen wird angezeigt.
  5. Die erforderlichen Daten in folgenden Feldern eingeben:
    • Unter Konfigurieren
    Feld Beschreibung
    Kamerastatus Auf Aktivieren klicken, um den Kamerastatus zu aktivieren.<> Oder<> Auf Deaktivieren klicken, um den Kamerastatus zu deaktivieren.
    Kameraname Geben Sie einen Namen für die Kamera ein.
    Kameratyp Aus der Dropdown-Liste den Kameratyp auswählen.
    Ermittelte IP-Kamera auswählen Aus der Dropdown-Liste die gewünschte IP-Kamera auswählen.
    Kameramodell Aus der Dropdown-Liste das Kameramodell auswählen.
    RTSP URL Die URL des Real Time Streaming Protocol (RTSP) Netzwerks eingeben.
    Benutzername Den Benutzernamen eingeben, um auf die URL des MPC Geräts zuzugreifen.
    Passwort Das Passwort eingeben, um auf die URL des MPC Geräts zuzugreifen.
    Abspielen Auf das Symbol klicken, um das auf der gewünschten Kamera vorhandene Clip abzuspielen.
    Zuordnungen Zum Zuordnen der Kamera auf irgendeine Tür klicken. Zum Beispiel: Tritt ein Ereignis wie Karte an Tür vorgefunden auf, beginnt die Kamera das Video aufzuzeichnen.
    DAUER DER CLIP AUFZEICHNUNG
    Vorereignis Auf die Zeit klicken, um festzulegen wir lange der Clip aufgezeichnet werden soll, bevor die Kamera eine Bewegung erkennt.
    Onsite-Nachereignis Auf die Zeit klicken, um festzulegen wir lange der Clip aufgezeichnet werden soll bis die Kamera, die Onsite angebracht ist, eine Bewegung erkennt.
    Offsite-Nachereignis Auf die Zeit klicken, um festzulegen wir lange der Clip aufgezeichnet werden soll bis die Kamera, die Offsite angebracht ist, eine Bewegung erkennt.
    EINSTELLUNGEN FÜR LIVE/AUFZEICHNUNG
    Bildrate Aus der Dropdown-Liste die Frequenz (Rate) auswählen mit der die Kamera einzigartige fortlaufende Bilder erfasst.
    Kameraauflösung Aus der Dropdown-Liste die Auflösung der Kamera auswählen. Sie ist vergleichbar mit der Anzahl der Pixel bei einem Bild einer Digitalkamera.
    Kameraqualität Aus der Dropdown-Liste die Kameraqualität auswählen.
    • Unter Ereignis aufzeichnen
    Feld Beschreibung
    Aufzeichnung eingeschaltet Aus der Liste mit vorhandenen Auslösern, den gewünschten Auslösepunkt auswählen, wann die Aufzeichnung starten soll.
    • Unter Daueraufzeichnung
    Feld Beschreibung
    Daueraufzeichnungs-Zeitplan Aus der Liste mit vorhandenen Zeitplänen, das Zeitfenster auswählen, in dem die Daueraufzeichnung aktiviert ist.
    • Unter Bewegung aufzeichnen
    Feld Beschreibung
    Bewegungsaufzeichnung aktivieren Zum Aktivieren der Bewegungsaufzeichnung auswählen.
    Haltezeit
    Bewegungsaufzeichnungs-Zeitplan Aus der Liste mit vorhandenen Zeitplänen, das Zeitfenster auswählen, in dem die Bewegungsaufzeichnung aktiviert ist.
    • Speichern klicken.
