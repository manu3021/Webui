Einstellungen bearbeiten

Im Reiter Einstellungen können Einstellungen hinzugefügt oder bearbeitet werden.

Zum Bearbeiten der Kontoeinstellungen:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Im linken Fensterbereich die gewünschten Entitäten auswählen, die bearbeitet werden müssen.
  3. Auf den Reiter Einstellungen klicken.
  4. Die erforderlichen Einstellungen bearbeiten, anschließend auf Einstellungen speichern klicken.
