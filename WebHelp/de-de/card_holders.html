Kartenhalter

Zum Erstellen eines Kartenhalters oder zum Hinzufügen neuer Karten:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Im linken Fensterbereich den Kunden auswählen, anschließend auf Personen doppelklicken. Die Seite Einstellungen wird angezeigt.
  3. Ein Kartenhalter und auch neue Karten können erstellt werden.

Zum Erstellen eines Kartenhalters:

  1. Auf der Seite Einstellungen auf Kartenhalter erstellen klicken. Die Seite KARTENHALTER ERSTELLEN wird angezeigt.
  2. Auf Bildgestaltung klicken, um nach einem Bild für den Kartenhalter zu suchen und eins hochzuladen.
  3. Unter BENUTZERINFORMATIONEN müssen Sie entsprechende Daten eingeben.
  4. Unter Zugeordnete Karten können Sie:
    • Eine neue Karte erstellen.
      • ODER
    • Aus verfügbaren Karten auswählen.

Zum Erstellen einer neuen Karte:

  1. Auf der Seite NEUE KARTE ERSTELLEN die folgenden Informationen eingeben:
    • Aus der Dropdown-Liste Einfach oder Mehrfach auswählen
    • Unter Kartenstatus den gewünschten Typ auswählen.
    • Unter dem Typ Revisionsstand die Revisionsnummer eingeben.
    • Unter Kartentyp, den Typ auswählen
    • Unter Anzahl der Anwendungen
    • Unter Kartennummer
    • Unter PIN
    • Unter vorläufig
    • Begrenzte Anzahl von Anwendungen
  2. Auf der Seite KARTENHALTER ERSTELLEN auf Speichern klicken.

Zum Auswählen aus verfügbaren Karten:

  1. Auf die Option Aus verfügbaren Karten auswählen klicken. Das Pop-up-Fenster VERFÜGBARE KARTEN wird angezeigt.
  2. Sie können in der Liste der verfügbaren Karten suchen und auswählen, anschließend auf Fertig klicken.
  3. Auf der Seite KARTENHALTER ERSTELLEN auf Speichern klicken.
