Einen Schnappschuss erstellen

Es kann ein Schnappschuss eines live oder aufgezeichneten Videobilds erfasst und als Bild verwendet werden. Sie können Schnappschüsse von einem aufgezeichneten Video erfassen.

Zum Erfassen eines Schnappschusses

  1. Im MPC Dashboard-Bereich auf den Reiter Viewer klicken.
  2. Es können ein oder mehrere Schnappschüsse mit folgenden Verfahren erfasst werden:
    • Zum Erfassen eines Schnappschusses: Einen Clip auswählen und Schnappschuss klicken. (ODER)
    • Zum Erfassen mehrerer Schnappschüsse: Im Reiter Viewer auf Schnappschuss klicken.
  3. Links im unteren Bereich des Bildschirms auf Schnappschuss klicken. Das Dialogfeld Speichern unter wird angezeigt.
  4. Einen Namen Name eingeben.
  5. Verzeichnis auswählen (den Ordner, wo der Schnappschuss gespeichert werden soll). Sie können auch über Durchsuchen ein anderes Verzeichnis auswählen.
  6. Das gewünschte Format (JPEG) für den Schnappschuss auswählen.
  7. Auf OK klicken, um den Schnappschuss zu speichern.
