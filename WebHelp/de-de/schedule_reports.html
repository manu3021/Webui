Berichte terminieren

Mit MPC können eine Reihe von Berichten erstellt werden. Diese Berichte können auf Basis von Filterkriterien erstellt werden. Berichte können terminiert, im Vorschau angezeigt und gedruckt werden.

So können Sie Berichte terminieren:

  1. Im MPC Dashboard-Bereich auf den Reiter Berichte klicken. Die Seite BERICHTE wird angezeigt. Es wird eine Liste mit allen Berichten mit Terminierung und Status angezeigt.
  2. In der Spalte Kartenstatus den gewünschten Bericht, der als Hyperlink angezeigt wird, anklicken.
  3. Die folgenden Funktionen ausführen, um die Filter für den Bericht zu sortieren und zu bestimmen.
    4. Feld Beschreibung
    Diesen Bericht terminieren
    Zeitplanformat Das Format für den terminierten Bericht auswählen.
    Ein
    Senden an
    Ausgewählte Benutzer Die Gesamtanzahl der Benutzer für die automatische Verteilung eingeben.
    Name des Berichts Einen Namen für Ihren Bericht eingeben.
  4. Bericht speichern klicken.

Hinweis:

Sie können auch das Format für den Bericht auswählen.
