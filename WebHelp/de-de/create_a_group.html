Eine Gruppe erstellen

Zum Erstellen einer Gruppe:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Im linken Fensterbereich das Kundenkonto auswählen.
  3. Auf das Symbol Erstellen klicken, anschließend Gruppe auswählen.Der Bildschirm NEUE GRUPPE wird angezeigt.
  4. Die erforderlichen Informationen über die Gruppe für das ausgewählte Konto eingeben und Speichern klicken.

Hinweis:

Es können auch unter einer bestehenden Gruppe weitere Gruppen angelegt werden.
