Kontoeinstellungen bearbeiten

Im Reiter Einstellungen können Kontoeinstellungen hinzugefügt oder bearbeitet werden.

Zum Bearbeiten der Kontoeinstellungen:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Im linken Fensterbereich auf das Konto Kunde/Händler klicken. Das Konto, das bearbeitet werden soll, auswählen.
  3. Auf den Reiter Einstellungen klicken.
  4. Die erforderlichen Einstellungen bearbeiten und auf Einstellungen speichern klicken.
