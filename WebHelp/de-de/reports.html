Berichte

Mit MPC können eine Reihe von Berichten erstellt werden. Diese Berichte können auf Basis von Filterkriterien erstellt werden. Berichte können in auf- oder absteigender Reihenfolge sortiert, in Vorschau angezeigt und gedruckt werden. Im Reiter Berichte kann Folgendes ausgeführt werden:

  • Berichte anzeigen: Diese Option in Liste Berichte listet alle vorhandenen Berichte auf, die Sie sich anzeigen lassen können.
  • Berichte erstellen: Mit der Option unter Neue Berichte können Sie neue Berichte erstellen, wie:

    • Audit
    • Karte
    • Ereignis
