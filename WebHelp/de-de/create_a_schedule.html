Zeitplan erstellen

Ein Zeitplan ist eine geplante Aufgabe, die in bestimmten Zeitabschnitten erledigt werden muss.

In MPC besitzt jedes Konto einen eindeutigen Zeitplan.

Aufgaben können so terminiert werden, dass sie zu bestimmter Zeit ausgeführt werden. Um eine Aufgabe für ein bestimmtes Konto zu terminieren, zuerst das Konto auswählen und dann den Zeitplan erstellen.

Zum Erstellen eines Zeitplans:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Zum Erstellen eines Zeitplans, müssen Sie im linken Fensterbereich eins aus dem Folgenden auswählen:
    • Händlerkonto
    • Kundenkonto
  3. Das Konto aufklappen, anschließend Terminieren, Feiertage klicken. Der Bildschirm Einstellungen wird angezeigt.
  4. Auf Zeitpläne klicken, anschließend die Option Neuen Zeitplan erstellen klicken. Die Seite ZEITPLAN wird angezeigt.
  5. Eine Bezeichnung für den Zeitplan in Name des Zeitplans und eine Kurz-Beschreibung eingeben
  6. Auf einen der Werktage klicken und den Mauszeiger auf die Endzeit des Zeitfensters ziehen.

    7. Tipp:

    Es reicht aus, das Zeitfenster für den Montag zu definieren. Für die restlichen Werktage kann dieses Zeitfenster mit Hilfe der Option Zeitplan kopieren kopiert werden.

  7. Die Zeitfenster für die Feiertage in Feiertag 1 (H1), Feiertag 2(H2), und Feiertag 3 (H3) einrichten.
  8. Speichern klicken.

Zum Bearbeiten eines Zeitplans:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Zum Erstellen eines Zeitplans, müssen Sie im linken Fensterbereich eins aus dem Folgenden auswählen:
    • Händlerkonto
    • Kundenkonto
  3. Das Konto aufklappen, anschließend Terminieren, Feiertage klicken. Der Bildschirm Einstellungen wird angezeigt.
  4. Auf Zeitpläne klicken, anschließend den Zeitplan, der bearbeitet werden soll, auswählen.
  5. Bearbeiten klicken. Die Seite ZEITPLAN wird angezeigt.
  6. Auf die Zeitplanleiste eines der Werktage klicken und den Mauszeiger auf die Endzeit des Zeitfensters ziehen.
  7. Speichern klicken.

Zum Löschen eines Zeitplans:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Zum Erstellen eines Zeitplans, müssen Sie im linken Fensterbereich eins aus dem Folgenden auswählen:
    • Händlerkonto
    • Kundenkonto
  3. Das Konto aufklappen, anschließend Terminieren, Feiertage klicken. Der Bildschirm Einstellungen wird angezeigt.
  4. Auf Zeitpläne klicken, anschließend den Zeitplan, der gelöscht werden soll, auswählen.
  5. Löschen klicken. Es wird eine Bestätigungsaufforderung angezeigt.
  6. Ja klicken.
