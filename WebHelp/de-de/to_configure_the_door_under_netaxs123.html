Zum Konfigurieren der Tür für NetAXS-123

Zum Konfigurieren der Türeinstellungen:

  1. Im MPC Dashboard-Bereich auf den Reiter Erstellen & Bearbeiten klicken. Die Seite ERSTELLEN UND BEARBEITEN wird angezeigt.
  2. Im linken Fensterbereich den Standort auswählen, anschließend das entsprechende Gerät, das dem Standort zugeordnet ist, klicken.

    3. Hinweis:

    Bevor Sie die Geräteeinstellungen bearbeiten, stellen Sie sicher, dass Sie die Geräte im Cloud registrieren.

  3. Im linken Fensterbereich auf das Symbol, das dem NetAXS-123-Gerät entspricht klicken, um alle verfügbaren Türen anzuzeigen.
  4. Auf die gewünschte Tür doppelklicken. Die Seite Einstellungen wird angezeigt.
  5. Klicken, um Folgendes zu bearbeiten:
    • Türname
    • Türverhalten
    • Zuordnungen
    • Zugeordneter Leser
    • Austritt-Schalter Einstellungen
  6. Speichern klicken.
