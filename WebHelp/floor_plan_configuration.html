Floor Plan Configuration

The Create and Edit tab enables you to configure the floor plan.

To configure the floor plan:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, you must select a site. The Maps tab displays the selected site location.
  3. Click the site icon present in the map to view the site floor plan.
  4. All the doors and cameras are listed on the left pane.To move the required cameras and doors to the floor plan, use the drag-and-drop operation.
  5. Click Save.
