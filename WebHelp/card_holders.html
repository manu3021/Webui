Card Holders

To create a card holder or add new cards:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, select the customer and then double-click People. The Settings page appears.
  3. You can create a card holder and also create new cards.

To create a card holder:

  1. In the Settings page, click Create Card Holder The CREATE CARD HOLDER page appears.
  2. Click the image layout to browse and upload the card holder's picture.
  3. Under USER DETAILS you must enter the relevant information.
  4. Under Associated Cards you can:
    • Create a new card
      • OR
    • Pick from available cards

To create a new card:

  1. In the CREATE NEW CARD page, enter the following details:
    • From the drop-down list, select Single or Multiple
    • Under Card Status select the required type.
    • Under Issue Level type the level of the issue.
    • Under Card Type, select the type of
    • Under Number of uses,
    • Under Card Number,
    • Under PIN
    • Under Temporary
    • Limited number of uses
  2. Click Save in the CREATE CARD HOLDER page.

To pick from available cards:

  1. Click the Pick from available cards option. The AVAILABLE CARDS pop-up window appears.
  2. You can search and select from the list of list of available cards and then click Done
  3. Click Save in the CREATE CARD HOLDER page.
