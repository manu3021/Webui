Edit the Settings

The Settings tab enables you to add or modify the settings.

To edit the account settings:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, click the required entities that must be edited.
  3. Click the Settings tab.
  4. Modify the required settings and then click Save Settings.
