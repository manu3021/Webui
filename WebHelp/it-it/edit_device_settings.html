Modifica Impostazioni Dispositivo

Per modificare le impostazioni HRRH8:

  1. Nell’area Dashboard MPC, fare clic sulla scheda Crea & Modifica. Viene visualizzata la pagina CREA E MODIFICA.
  2. Nel riquadro di sinistra, selezionare il sito, quindi fare doppio clic sul dispositivo pertinente aggiunto al sito. Viene visualizzata la pagina Impostazioni.

    3. Nota:

    Prima di modificare le impostazioni del dispositivo, verificare di registrare i dispositivi su cloud.

  3. Inserire i dettagli registrati nei seguenti campi.
    • Alla voce CONFIGURAZIONE DI RETE,
      • Chiave univoca: Digitare l’MACID del dispositivo. Il MACID deve essere ottenuto dal pannello.
      • Indirizzo IP
      • Maschera di sottorete
      • Server DNS
      • Porta HTTP
      • Gateway
    • Alla voce IMPOSTAZIONI DEL REGISTRATORE,
      • Nome del registratore
      • Versione hardware
      • Formato video
      • Indirizzo Mac
      • Versione del firmware
    • Alla voce IMPOSTAZIONI MAXPRO CONNECT,
      • Attiva: Per modificare l’URL del server.
    • Alla voce IMPOSTAZIONI AVANZATE,
      • Ripristina password
      • Aggiorna password
    • Per configurare le impostazioni della telecamera HRRH8, rimandiamo a Per configurare la telecamera in HRRH8

Per modificare le impostazioni NetAXS-123:

  1. Nell’area Dashboard MPC, fare clic sulla scheda Crea & Modifica. Viene visualizzata la pagina CREA E MODIFICA.
  2. Nel riquadro di sinistra, selezionare il sito, quindi fare doppio clic sul dispositivo pertinente aggiunto al sito. Viene visualizzata la pagina Impostazioni.

    3. Nota:

    Prima di modificare le impostazioni del dispositivo, verificare di registrare i dispositivi su cloud.

  3. Inserire i dettagli registrati nei seguenti campi.
    • Nome di comando del pannello
    • Descrizione
    • Comunicazione
    • Indirizzo pannello
    • Comando firmware
    • Intervallo di polling comando I/O
    • Loop intervallo di verifica
    • Ripetizione conteggio comando pannello
    • Timeout comando pannello
    • Fuso orario della zona del comando
    • Impostare data e ora
    • Anti-Passback: Locale, Dimentica e Globale.
    • Costrizione
    • Inverti LED
    • Lettura continua
    • Ingressi uscita liberi
  4. Fare clic su Salva.

  • Per configurare le impostazioni di ingresso NetAXS-123, rimandiamo a Per configurare gli ingressi per NetAXS-123
  • Per configurare le impostazioni di uscita NetAXS-123, rimandiamo a Per configurare le uscite per NetAXS-123
  • Per configurare le impostazioni del Lettore NetAXS-123, rimandiamo a Per configurare il Lettore sotto la voce NetAXS123
