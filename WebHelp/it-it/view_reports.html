Visualizza i report

L’opzione Visualizza Report consente di visualizzare i seguenti report:

  • Revisione
  • Tessera
  • Evento

Per visualizzare un report:

  1. Nell’area Dashboard MPC, fare clic sulla scheda Report.
  2. Fare clic su Visualizza Report. Viene visualizzata la pagina REPORT.
  3. Sotto la colonna Elenco Report, fare clic sull’hyperlink report richiesto. Ad esempio, selezionare REPORT TESSERE.
  4. Eseguire le seguenti funzioni.
