Report

Sarà possibile generare diversi report servendosi di MPC. Questi report possono essere generati sulla base dei criteri del filtro. I report possono essere ordinati in ordine ascendente o discendente e possono essere visualizzati in anteprima e stampati La scheda Report consente di eseguire le seguenti operazioni:

  • Visualizza Report: L’opzione sotto la voce Elenco Report consente di visualizzare un elenco di tutti i report disponibili.
  • Creare report: L’opzione sotto la voce Nuovo report consente di creare nuovi report quali ad esempio:

    • Revisione
    • Tessera
    • Evento
