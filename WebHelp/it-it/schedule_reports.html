Programma report

Sarà possibile generare diversi report servendosi di MPC. Questi report possono essere generati sulla base dei criteri del filtro. I report possono essere programmati, visti in anteprima e stampati.

Come programmare i report:

  1. Nell’area Dashboard MPC, fare clic sulla scheda REPORT. Viene visualizzata la pagina REPORT. Viene visualizzato un elenco di tutti i report, con programma e stato.
  2. Sotto la voce Report Tessere fare clic sul report richiesto, che viene visualizzato come hyperlink.
  3. Eseguire le seguenti funzioni per ordinare e impostare i filtri per il report.
    4. Campo Descrizione
    Programma questo Report
    Formato programma Selezionare il tipo di formato per il report programmato.
    On
    Invia a
    Utenti selezionati Digitare il numero totale di utenti per l’invio dei report automatici.
    Attribuire un nome al Report Digitare un nome per il proprio report
  4. Fare clic su Salva Report.

Nota:

Sarà inoltre possibile selezionare un Tipo di Formato per il report.
