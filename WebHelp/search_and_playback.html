Search and Playback

The Clip feature enables you to search for recorded video from a particular camera.

To view a clip:

  1. In the MPC Dashboard area, click the Viewer tab.
  2. Click the icon. The Universal List pane appears.
  3. Under the Universal List, select a single camera.

    4. Note:

    • Under the Universal List, click Clip Search to select and search multiple cameras.
    • To select all the cameras under site, select the site check-box.
  4. Drag and drop the camera to the viewer pane to view a live video.
  5. Click Clip at the lower left of the screen. The option for an instant single camera search appears.
  6. In the clip search window, click the icon. You can now filter the search for recorded videos, based on search conditions like video recorded today, yesterday, and custom.
  7. Select one of the following search conditions corresponding to date time filter.
    • Today: Video recorded in MPC today.
    • Yesterday: Video recorded in MPC yesterday.
    • Custom: You can customize the search.
  8. Select one of the following for the clip storage and clip type:
    • Clip type
      • Motion Event
      • Schedule
      • Trigger Event
      • User Trigger
      • Rule Driven
    • Clip storage
      • Onsite
      • Oncloud

    9. Note:

    Schedule type is not supported by Oncloud storage.

  9. Click Search.
