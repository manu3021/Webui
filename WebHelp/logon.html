Log On

To log on to the MPC user interface:

  1. Type the Maxpro Cloud User Interface URL on your browser. The MAXPRO Cloud Signin page appears.
  2. You can log on using the following options:
    • As a registered user
    • As a new user

The following table displays the log on procedures for new and registered users.

If you are a registered user... If you are a new user...
a. Type the User Name and Password

Note

The User Name is the e-mail ID provided during the registration.

b. Click SIGN IN.

Note

Before you sign up, you must ensure to obtain the sign up and the approval details from the dealer.

a. Click Create new Account The Dealer Signup page appears.
b. Enter the required details and click Sign Up

After you receive the new account registration and account activation mail, you can continue with the sign-up.

Note:

If you are unable to log on, click Get Help? to reset your password.
