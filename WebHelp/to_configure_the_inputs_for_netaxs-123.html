To Configure the Inputs for NetAXS-123

To configure the input settings:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, select the site and then click the relevant device added to the site.

    3. Note:

    Before you modify the device settings, ensure to register the devices on cloud.

  3. In the left pane, click the icon corresponding to the NetAXS-123 device to expand and view all the available doors.
  4. Double-click the required door to expand and view all the associated inputs. The Settings page appears.
  5. Enter the required details in the following fields:
    6. Field Description
    Input Name Enter a unique name to identify the door inputs.
    Shunt Time Zone Specifies the time period during which the input will be ignored.
    Block Interlock Time Zone Enables you to block the interlock time zone, or programmed interaction between two points.
    Disable Alarms and Events Specifies the time period during which Alarm and events will not be reported.
    Shunt Time Specifies the amount of time for which the inputs will be shunted, or de-activated.
    Debounce Time Specifies the period of time the input must remain in a new state before generating an alarm.
    Normally Normally Closed – Specifies that the input’s normal state is closed (default).
    Normally Open – Specifies that the input’s normal state is open.
    Supervised Specifies that the input’s electrical circuit is wired with alternative paths supervised by resistors.
    Auto-Relock Causes the door to re-lock immediately when the door status switch closes after entry. The output relay that controls the door strike de-energizes when the associated input returns to normal state instead of remaining energized for the duration of the pulse time.
    Interlocking Enables you to disable the interlock, or programmed interaction between two points. When enabled, this point ignores all interlock actions to it.
  6. Click Save.
