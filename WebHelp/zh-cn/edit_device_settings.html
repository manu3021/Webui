编辑设备设置

要编辑 HRRH8 设置：

  1. 在“MPC 仪表板”区中，单击创建和编辑选项卡。随即显示创建和编辑页面。
  2. 在左窗格中选择地点，然后双击添加到此地点的相关设备。随即显示设置页面。

    3. 注：

    修改设备设置之前，务必在云端注册相关设备。

  3. 在以下字段中输入所要求的详情：
    • 在 网络配置下：
      • 唯一密钥：键入设备的 MACID。MACID 必须从面板上获得。
      • IP 地址
      • 子网掩码
      • DNS 服务器
      • HTTP 端口
      • 网关
    • 在 录像机设置下：
      • 录像机名称
      • 硬件版本
      • 视频格式
      • Mac 地址
      • 固件版本
    • 在 MAXPRO 连接设置下：
      • 启用：以修改服务器的 URL。
    • 在 高级设置下：
      • 恢复密码
      • 更新密码
    • 要配置 HRRH8 摄像头设置，请参考 配置 HRRH8 摄像头

要编辑 NetAXS-123 设置：

  1. 在“MPC 仪表板”区中，单击创建和编辑选项卡。随即显示创建和编辑页面。
  2. 在左窗格中选择地点，然后双击添加到此地点的相关设备。随即显示设置页面。

    3. 注：

    修改设备设置之前，务必在云端注册相关设备。

  3. 在以下字段中输入所要求的详情：
    • 面板名称
    • 描述
    • 沟通
    • 面板地址
    • 固件
    • I/O 轮询间隔
    • 循环验证间隔
    • 面板CMD重试计数
    • 面板 CMD 超时
    • 地区时区
    • 设置日期时间
    • 防潜回：局部、宽容和全局。
    • 强制
    • 反向 LED
    • 连续读取
    • 自由出口输入
  4. 单击保存。
  • 要配置 NetAXS-123 门户设置，请参考 配置 NetAXS-123 门户
  • 要配置 NetAXS-123 输入设置，请参考 配置 NetAXS-123 输入
  • 要配置 NetAXS-123 输出设置，请参考 配置 NetAXS-123 输出
  • 要配置 NetAXS-123 阅读器设置，请参考 配置 NetAXS-123 阅读器
