配置 HRRH8 摄像头

要配置摄像头设置：

  1. 在“MPC 仪表板”区中，单击创建和编辑选项卡。随即显示创建和编辑页面。
  2. 在左窗格中选择地点，然后单击添加到此地点的相关设备。

    3. 注：

    修改设备设置之前，务必在云端注册相关设备。

  3. 在左窗格中，单击 图标（此图标对应于 HRRH8 设备），以展开并查看所有可用摄像头。
  4. 双击所要求的摄像头。随即显示设置页面。
  5. 在以下字段中输入所要求的详情：
    • 在配置下：
    字段 描述
    摄像头状态 单击启用，以启用摄像头状态。<> 或<> 单击禁用，以禁用摄像头状态。
    摄像头名称 为摄像头输入一个名称。
    摄像头类型 从下拉列表中选择摄像头的类型。
    选择已发现的 IP 摄像头 从下拉列表中选择所要求的 IP 摄像头。
    摄像头型号 从下拉列表中选择摄像头的型号。
    RTSP URL 输入“实时流协议 (RTSP)”网络的 URL。
    用户名 输入用于访问 MPC 装置的 URL 的用户名。
    密码 输入用于访问 MPC 装置的 URL 的密码。
    播放 单击 图标可播放所选摄像头中的可用剪辑。
    关联 单击可将摄像头与任何门户关联起来。例如，可以设置当在门户处发生任何事件（比如发现卡）时，摄像头开始录制视频。
    录制剪辑持续时间
    前期事件 单击此时间可设置摄像头在检测到运动之前的剪辑录制时间。
    场内后期事件 单击此时间可设置场内摄像头在检测到运动之后的剪辑录制时间。
    场外后期事件 单击此时间可设置场外摄像头在检测到运动之后的剪辑录制时间。
    实时/录制设置
    帧速率 从下拉列表中，选择摄像头将以多快的频率（速率）创建唯一连续图像（这样的图像称为“帧”）。
    摄像头分辨率 从下拉列表中，选择摄像头分辨率。这是指数字摄像头图像的总像素数。
    摄像头品质 从下拉列表中，选择摄像头的品质。
    • 在事件录制下：
    字段 描述
    录制 从可用触发器列表中，选择所需的触发点，即开始录制的时间。
    • 在连续录制下：
    字段 描述
    连续录制时间表 从可用时间表列表中，选择所要求的期限。在这个期限中将启用连续录制。
    • 在运动录制下：
    字段 描述
    启用运动录制 选择此项可启用运动录制。
    保持时间
    运动录制时间表移动 从可用时间表列表中，选择所要求的期限。在这个期限中将启用运动录制。
    • 单击保存。
