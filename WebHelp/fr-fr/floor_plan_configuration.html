Configuration du plan des étages

L'onglet Créer et modifier vous permet de configurer le plan d'étages.

Pour configurer le plan d'étages :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, vous devez sélectionner un site. L'onglet Cartes affiche l'emplacement du site sélectionné.
  3. Cliquez sur l'icône de site sur la carte pour visualiser le plan d'étages du site.
  4. Toutes les portes et caméras sont listées dans le volet gauche. Pour déplacer les caméras et les portes souhaitées vers le plan d'étages, faites un glisser-déposer.
  5. Cliquez sur Enregistrer.
