Créer un périphérique

Pour créer un périphérique :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, vous devez sélectionner un site.
  3. Cliquez sur l'icône créer et cliquez sur Créer>Périphérique. L'écran NOUVEAU PÉRIPHÉRIQUE apparaît.

    4. Remarque :

    vous pouvez créer un périphérique uniquement dans un site.

  4. Sélectionnez les Types de périphérique dans la liste déroulante. Vous pouvez sélectionner HRRH8 ou Panneau NS123.
  5. Entrez la Clé unique (MAC ID) du périphérique sélectionné.
  6. Cliquez sur Enregistrer.

Vous pouvez modifier les paramètres du périphérique.

  • Pour modifier les paramètres du périphérique, cliquez sur Modifier les paramètres du périphérique.
