Gestion des alarmes

L'illustration suivante montre l'onglet Alarmes.

Les éléments suivants s'affichent à l'écran.

N° Champ Description
1 Recherche Les alarmes informent les opérateurs de la survenue d'événements. L'option de recherche vous permet de chercher dans la liste des alarmes reposant sur :<> a. Caméras<> b. Portes<> c. Règles<> d. Temps
2 Actions Vous permet d'exécuter les actions suivantes pour toutes les alarmes apparaissant dans la fenêtre des Alarmes. <> a. Acquitter : Acquittez l'alarme sélectionnée.<> b. Effacer : Effacez automatiquement les alarmes lorsqu'elles sont acquittées.<> c. Bloquer le flux d'alarmes : Arrêtez de recevoir des alarmes récentes ou acquittées dans la fenêtre des Alarmes.<> L'état de l'alarme, après son déclenchement, est qualifié de non acquitté.
3 Vues Vous permet de sélectionner les options demandées devant apparaître sous forme d'affichage à colonnes.
4 Acquittement Vous permet d'acquitter une alarme pour accepter l'intervention nécessaire entreprise.
