Rapports

Vous pouvez générer de nombreux rapports à l'aide de MPC. Ces rapports peuvent être générés en s'appuyant sur les critères de filtre. Les rapports peuvent être triés dans un ordre croissant ou décroissant et peuvent être prévisualisés et imprimés. L'onglet Rapports vous permet d'exécuter les actions suivantes :

  • Afficher les rapports : Cette option dans Liste des rapports vous permet d'afficher une liste de tous les rapports disponibles.
  • Créer des rapports : L'option dans Nouveau rapport vous permet de créer des nouveaux rapports, par exemple :

    • Audit
    • Carte
    • Événement
