Configurer la caméra dans HRRH8

Pour configurer les paramètres de caméra :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, sélectionnez le site et cliquez sur le périphérique correspondant ajouté au site.

    3. Remarque :

    avant de modifier les paramètres de périphérique, assurez-vous d'enregistrer les périphériques dans le cloud.

  3. Dans le volet gauche, cliquez sur l'icône correspondant au périphérique HRRH8 pour afficher toutes les caméras disponibles.
  4. Double-cliquez sur la caméra souhaitée. La page Paramètres apparaît.
  5. Entrez les informations demandées dans les champs suivants :
    • Dans Configurer
    Champ Description
    État de la caméra Cliquez sur Activer pour activer l'état de la caméra.<> Ou<> Cliquez sur Désactiver pour désactiver l'état de la caméra.
    Nom de la caméra Entrez un nom de caméra.
    Type de caméra Dans la liste déroulante, sélectionnez le type de caméra.
    Sélectionnez Caméra IP découverte Dans la liste déroulante, sélectionnez la caméra IP souhaitée.
    Modèle de caméra Dans la liste déroulante, sélectionnez le modèle de caméra.
    URL RTSP Entrez l'URL du protocole de diffusion multimédia en temps réel (RTSP).
    Nom d'utilisateur Entrez le nom d'utilisateur pour accéder à l'URL du périphérique MPC.
    Mot de passe Entrez le mot de passe pour accéder à l'URL du périphérique MPC.
    Lecture Cliquez sur l'icône pour lire le clip disponible dans la caméra sélectionnée.
    Associations Cliquez pour associer la caméra à n'importe quelle porte. Ainsi, en cas d'événement concernant la porte, p. ex. une carte trouvée, la caméra commence l'enregistrement de la vidéo.
    DURÉE D'ENREGISTREMENT DU CLIP
    Pré-événement Cliquez sur temps pour définir la durée d'enregistrement du clip avant la détection du mouvement par la caméra.
    Post-événement sur le site Cliquez sur temps pour définir la durée d'enregistrement du clip après la détection du mouvement par la caméra, située sur le site.
    Post-événement hors site Cliquez sur temps pour définir la durée d'enregistrement du clip après la détection du mouvement par la caméra, située hors site.
    PARAMÈTRE D'UNE VIDÉO EN DIRECT/ENREGISTRÉE
    Fréquence de trame Dans la liste déroulante, sélectionnez la fréquence (taux) à laquelle la caméra capture des images consécutives uniques appelées trames.
    Résolution de la caméra Dans la liste déroulante, sélectionnez la résolution de la caméra. Il s'agit du nombre total de pixels dans une image numérique (appareil photo/caméra).
    Qualité de la caméra Dans la liste déroulante, sélectionnez la qualité de la caméra.
    • Dans Enregistrement d'événements
    Champ Description
    Enregistrer à Dans la liste des déclencheurs disponibles, sélectionnez le point déclencheur souhaité marquant le début d'enregistrement.
    • Dans Enregistrement continu
    Champ Description
    Programmation de l'enregistrement continu Dans la liste des programmations disponibles, sélectionnez la période souhaitée pendant laquelle l'enregistrement est activé.
    • Dans Enregistrement des mouvements
    Champ Description
    Activer l'enregistrement des mouvements Sélectionnez pour activer l'enregistrement des mouvements.
    Temps de maintien
    Programmation de l'enregistrement des mouvements Dans la liste des programmations disponibles, sélectionnez la période souhaitée pendant laquelle l'enregistrement des mouvements est activé.
    • Cliquez sur Enregistrer.
