Se connecter

Pour se connecter à l'interface utilisateur MPC :

  1. entrez l'URL de l'interface utilisateur Maxpro Cloud dans votre navigateur. La page Connexion MAXPRO Cloud apparaît.
  2. Vous pouvez vous connecter à l'aide des options suivantes :
    • En tant qu'utilisateur enregistré
    • En tant que nouvel utilisateur

Le tableau suivant affiche les procédures de connexion pour les nouveaux utilisateurs et les utilisateurs enregistrés.

Si vous êtes un utilisateur enregistré... Si vous êtes un nouvel utilisateur...
a. Entrez le nom d'utilisateur et le mot de passe

Remarque

Le Nom d'utilisateur est l'identifiant de messagerie fourni pendant l'inscription.

b. Cliquez sur CONNEXION.

Remarque

Avant de vous inscrire, vous devez être sûr d'obtenir les détails de l'inscription et de l'autorisation de la part du distributeur.

a. Cliquez sur Créer un nouveau compte La page Inscription du distributeur s'affiche.
b. Entrez les informations requises et cliquez sur S'inscrire

Après avoir reçu l'enregistrement du nouveau compte et l'e-mail d'activation de celui-ci, vous pouvez continuer l'inscription.

Remarque :

si vous n'arrivez pas à vous connecter, cliquez sur Aide? pour réinitialiser votre mot de passe.
