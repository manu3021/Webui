Configurer les entrées pour NetAXS-123

Pour configurer les paramètres d'entrée :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, sélectionnez le site et cliquez sur le périphérique correspondant ajouté au site.

    3. Remarque :

    avant de modifier les paramètres de périphérique, assurez-vous d'enregistrer les périphériques dans le cloud.

  3. Dans le volet gauche, cliquez sur l'icône correspondant au périphérique NetAXS-123 pour afficher toutes les portes disponibles.
  4. Double-cliquez sur la porte souhaitée pour afficher toutes les entrées associées. La page Paramètres apparaît.
  5. Entrez les informations demandées dans les champs suivants :
    6. Champ Description
    Nom d'entrée Entrez un nom unique pour identifier les entrées de porte.
    Zone horaire de shunt Spécifie la période pendant laquelle l'entrée sera ignorée.
    Bloquer la période d'enclenchement Vous permet de bloquer la période d'enclenchement ou l'interaction programmée entre deux points.
    Désactiver des alarmes et des événements Spécifie la période pendant laquelle les alarmes et les événements ne seront pas rapportés.
    Période de shunt Spécifie la durée pendant laquelle les entrées seront shuntées ou désactivées.
    Temps de réponse Spécifie la période pendant laquelle l'entrée doit rester dans son état avant de générer une alarme.
    Normalement Normalement fermé - Spécifie que l'état normal de l'entrée est fermé (par défaut).<> Normalement ouvert - Spécifie que l'état normal de l'entrée est ouvert.
    Surveillé Spécifie que le circuit électrique de l'entrée est raccordé à des voies alternatives surveillées par des résistances électriques.
    Reverrouillage automatique Provoque le reverrouillage automatique de la porte lorsque le commutateur d'état de celle-ci se ferme après l'entrée. Le relais de sortie, qui commande la gâche de la porte, est désactivé lorsque l'entrée associée retourne à son état normal au lieu de rester sous tension pendant la durée de l'impulsion.
    Interverrouillage Vous permet de désactiver l'interverrouillage ou l'interaction programmée entre deux points. En cas d'activation, ce point ignore toutes les actions d'interverrouillage sur lui.
  6. Cliquez sur Enregistrer.
