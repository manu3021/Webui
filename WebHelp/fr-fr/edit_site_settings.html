Modifier les paramètres du site

Pour modifier les paramètres du site :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, sélectionnez le site. La page Paramètres apparaît.
  3. Double-cliquez sur la trame de pointillés pour charger le plan des étages du site sélectionné.

    4. Remarque :

    vous pouvez charger uniquement les plans des étages qui sont au format .JPG ou .PNG.

  4. Entrez les informations requises et cliquez sur Enregistrer.
