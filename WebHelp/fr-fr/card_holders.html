Détenteurs de carte

Pour créer un détenteur de carte ou ajouter de nouvelles cartes :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, sélectionnez le client et double-cliquez sur Personnes. La page Paramètres apparaît.
  3. Vous pouvez créer un détenteur de carte ainsi que de nouvelles cartes.

Pour créer un détenteur de carte :

  1. dans la page Paramètres, cliquez sur Créer un détenteur de carte. La page CRÉER UN DÉTENTEUR DE CARTE apparaît.
  2. Cliquez sur la disposition d'image pour naviguer et charger la photo du détenteur de la carte.
  3. Vous devez entrer les informations appropriées dans DONNÉES UTILISATEUR.
  4. Dans Cartes Associées, vous pouvez :
    • Créer une nouvelle carte
      • OU
    • Choisir parmi les cartes disponibles

Pour créer une nouvelle carte :

  1. Dans la page CRÉER UNE NOUVELLE CARTE, entrez les informations suivantes :
    • Dans la liste déroulante, sélectionnez Unique ou Multiples
    • Dans État de la carte, sélectionnez le type requis.
    • Dans Niveau de délivrance, saisissez le niveau de la délivrance.
    • Dans Type de carte, sélectionnez le type
    • Dans Nombre d'utilisations,
    • Dans Numéro de carte,
    • Dans PIN
    • Dans Provisoire
    • Nombre limité d'utilisations
  2. Cliquez sur Enregistrer sur la page CRÉER UN DÉTENTEUR DE CARTE.

Choisir parmi les cartes disponibles :

  1. Cliquez sur l'option Choisir parmi les cartes disponibles. La fenêtre pop-up CARTES DISPONIBLES apparaît.
  2. Vous pouvez rechercher et sélectionner une carte dans la liste des cartes disponibles, puis cliquez sur Terminé.
  3. Cliquez sur Enregistrer sur la page CRÉER UN DÉTENTEUR DE CARTE.
