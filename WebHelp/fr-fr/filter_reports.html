Filtrer des rapports

Vous pouvez générer de nombreux rapports à l'aide de MPC. Ces rapports peuvent être générés en s'appuyant sur les critères de filtre. Les rapports peuvent être programmés, prévisualisés et imprimés.

Pour filtrer des rapports :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Rapports. La page RAPPORTS apparaît. Une liste de tous les rapports, avec la programmation et l'état, apparaît.
  2. Dans la colonne Rapports cartes, cliquez sur le rapport demandé qui apparaît comme hyperlien.
  3. Exécutez les fonctions suivantes pour trier et définir les filtres pour le rapport.
    4. Champ Description
    Compte Dans la liste des comptes disponibles, sélectionnez le compte demandé.
    État de la carte Dans la liste des états de carte disponibles, sélectionnez l'état demandé.
    Nom du détenteur de carte Entrez le nom du détenteur de carte à filtrer.
    Accès aux personas Entrez le nombre d'accès aux personas associés au détenteur de la carte.
    Porte/lecteurs Entrez le nombre de porte/lecteurs associés au détenteur de la carte.
    Sélectionner une date
    Date d’activation
    Date de début Sélectionnez une date pour activer le rapport.
    Date d'expiration Sélectionnez une date pour l'expiration du rapport.
    Sélection des colonnes
    Sélectionner des colonnes Sélectionnez les colonnes à montrer dans l'affichage des rapports.
  4. Cliquez sur Prévisualiser.
