Fermer la session

Vous pouvez vous déconnecter de MPC dans le menu Utilisateur.

Pour se déconnecter de MPC

  1. Dans l'espace tableau de bord MPC, cliquez sur le coin en haut à droite. Les options du menu utilisateur s'affichent.
  2. Cliquez sur Déconnexion.
