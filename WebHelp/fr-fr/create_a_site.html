Créer un site

Les comptes sont définis en ajoutant un site.

Pour créer un site :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, sélectionnez le compte client.
  3. Cliquez sur l'icône créer, puis sur Site. L'écran NOUVEAU SITE apparaît.
  4. Entrez les informations requises du site pour le compte sélectionné et cliquez sur Enregistrer.

Remarque :

vous pouvez cliquer sur Naviguer et charger le plan pour naviguer et charger le plan sur le site. Les formats de fichier pris en charge sont JPEG et PNG.

Pour en savoir plus sur la manière de configurer le plan, référez-vous à la Configuration du plan des étages.

Vous pouvez modifier les paramètres du site et de service du site.

  • Pour modifier les paramètres du site, cliquez sur Modifier les paramètres du site.
  • Pour modifier les paramètres de service du site, cliquez sur Modifier les paramètres de service du site

Vous pouvez mettre à niveau le microprogramme.

  • Pour mettre à niveau le microprogramme, cliquez sur Mise à niveau du microprogramme.
