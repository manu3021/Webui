Supprimer

Pour supprimer un client, site groupe ou périphérique :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet CRÉER ET MODIFIER. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, sélectionnez le client, site, groupe ou périphérique, puis cliquez sur Supprimer. Un message d'avertissement apparaît. Vous pouvez supprimer les éléments suivants :
    • Client
    • Site
    • Groupe
    • Périphérique

    3. Après avoir sélectionné un client, site, groupe ou périphérique demandé, un message de confirmation apparaît.

  3. Cliquez sur Oui.

Remarque :

toutes les entités dans le nœud parent sont supprimées.
