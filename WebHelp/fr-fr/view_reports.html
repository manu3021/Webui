Afficher des rapports

L'option Afficher des rapports vous permet d'afficher les rapports suivants :

  • Audit
  • Carte
  • Événement

Pour afficher un rapport :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Rapports.
  2. Cliquez sur Afficher des rapports. La page RAPPORTS apparaît.
  3. Dans la colonne Liste des rapports, cliquez sur l'hyperlien du rapport demandé. Par exemple, sélectionnez RAPPORTS DE CARTES.
  4. Exécutez les fonctions suivantes.
