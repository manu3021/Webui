Modifier les paramètres de compte

L'onglet Paramètres vous permet d'ajouter ou de modifier les paramètres de compte.

Pour modifier les paramètres de compte :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, cliquez sur le compte client/distributeur. Sélectionnez le compte à modifier.
  3. Cliquez sur l'onglet Paramètres.
  4. Modifiez les paramètres demandés et cliquez sur Enregistrer les paramètres.
