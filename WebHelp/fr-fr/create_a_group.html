Créer un groupe

Pour créer un groupe :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, sélectionnez le compte client.
  3. Cliquez sur l'icône créer et sélectionnez Groupe. L'écran NOUVEAU GROUPE apparaît.
  4. Entrez les informations requises du groupe pour le compte sélectionné et cliquez sur Enregistrer.

Remarque :

vous pouvez inclure également un groupe dans un groupe existant.
