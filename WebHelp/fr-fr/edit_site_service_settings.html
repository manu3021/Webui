Modifier les services du site

Pour modifier les services du site :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, sélectionnez le site.
  3. Cliquez sur l'onglet Services. Les détails du programme de périphériques apparaissent.
  4. Sélectionnez la Durée de conservation dans l'option disponible. La durée de conservation est le nombre de jours pendant lesquels les clips vidéo sont disponibles dans le cloud.
  5. Sélectionnez le programme demandé dans la liste des options disponibles.
