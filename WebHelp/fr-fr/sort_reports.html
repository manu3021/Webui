Trier des rapports

Vous pouvez générer de nombreux rapports à l'aide de MPC. Ces rapports peuvent être générés en s'appuyant sur les critères de filtre. Les rapports peuvent être programmés, prévisualisés et imprimés.

Pour trier des rapports :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet RAPPORTS. La page RAPPORTS apparaît. Une liste de tous les rapports, avec la programmation et l'état, apparaît.
  2. Dans la colonne Rapports cartes, cliquez sur le rapport demandé qui apparaît comme hyperlien.
  3. Exécutez les fonctions suivantes pour définir les filtres pour le rapport.
    4. Champ Description
    Tri
    Ordre de tri1 Définissez le premier type d'ordre suivant lequel vous devez filtrer le rapport.
    Ordre de tri2 Définissez le second type d'ordre suivant lequel vous devez filtrer le rapport.
    Ordre de tri3 Définissez le troisième type d'ordre suivant lequel vous devez filtrer le rapport.
  4. Cliquez sur Prévisualiser.
