Programmer des rapports

Vous pouvez générer de nombreux rapports à l'aide de MPC. Ces rapports peuvent être générés en s'appuyant sur les critères de filtre. Les rapports peuvent être programmés, prévisualisés et imprimés.

Comment programmer des rapports :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet RAPPORTS. La page RAPPORTS apparaît. Une liste de tous les rapports, avec la programmation et l'état, apparaît.
  2. Dans la colonne Rapports cartes, cliquez sur le rapport demandé qui apparaît comme hyperlien.
  3. Exécutez les fonctions suivantes pour trier et définir les filtres pour le rapport.
    4. Champ Description
    Programmer ce rapport
    Format de programmation Sélectionnez le type de format du rapport programmé.
    Date/heure
    Envoyer à
    Utilisateurs sélectionnés Entrez le nombre total d'utilisateurs pour l'envoi automatique de rapports.
    Nommez votre rapport Entrez un nom pour votre rapport.
  4. Cliquez sur Enregistrer le rapport.

Remarque :

vous pouvez aussi sélectionner un type de format pour le rapport.
