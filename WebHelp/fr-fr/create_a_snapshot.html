Créer un instantané

Vous pouvez prendre un instantané d'une vidéo en direct ou enregistrée et l'utiliser comme image. Vous pouvez créer des instantanés de la vidéo enregistrée.

Pour créer un instantané

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Visionneuse.
  2. Vous pouvez prendre un ou plusieurs instantanés selon les méthodes suivantes :
    • Pour prendre un instantané : Sélectionnez un clip vidéo et cliquez sur Instantané. (OU)
    • Pour prendre plusieurs instantanés : dans l'onglet Visionneuse, cliquez sur Instantané.
  3. Cliquez sur Instantané en bas à gauche de l'écran. La boîte de dialogue Enregistrer sous apparaît.
  4. Entrez un Nom.
  5. Sélectionnez l'Emplacement (dossier dans lequel vous souhaitez enregistrer l'instantané). Vous pouvez aussi Naviguer et sélectionner un emplacement de dossier différent.
  6. Sélectionnez le Format requis (JPEG) pour l'instantané.
  7. Cliquez sur OK pour enregistrer l'instantané.
