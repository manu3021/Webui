Modifier les paramètres du périphérique

Pour modifier les paramètres de HRRH8 :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, sélectionnez le site et double-cliquez sur le périphérique correspondant ajouté au site. La page Paramètres apparaît.

    3. Remarque :

    avant de modifier les paramètres de périphérique, assurez-vous d'enregistrer les périphériques dans le cloud.

  3. Entrez les informations demandées dans les champs suivants :
    • Dans CONFIGURATION DE RÉSEAU,
      • Clé unique : entrez l'ID MAC du périphérique. L'ID MAC se trouve dans le panneau.
      • Adresse IP
      • Masque de sous-réseau
      • Serveur DNS
      • Port HTTP
      • Passerelle
    • Dans PARAMÈTRES DE L'ENREGISTREUR,
      • Nom de l'enregistreur
      • Version matérielle
      • Format vidéo
      • Adresse Mac
      • Version du microprogramme
    • Dans PARAMÈTRES MAXPRO CONNECT ,
      • Activer : pour modifier l'URL du serveur.
    • Dans PARAMÈTRES AVANCÉS,
      • Restaurer le mot de passe
      • Actualiser le mot de passe
    • Pour configurer les paramètres de caméra HRRH8, référez-vous à Configurer la caméra dans HRRH8

Pour modifier les paramètres NetAXS-123 :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, sélectionnez le site et double-cliquez sur le périphérique correspondant ajouté au site. La page Paramètres apparaît.

    3. Remarque :

    avant de modifier les paramètres de périphérique, assurez-vous d'enregistrer les périphériques dans le cloud.

  3. Entrez les informations demandées dans les champs suivants :
    • Nom du panneau
    • Description
    • Communication
    • Adresse du panneau
    • Microprogramme
    • Intervalle d'interrogation E/S
    • Intervalle de vérification de boucle
    • Nombre de tentatives de CMD au panneau
    • Délai d'attente de CMD au panneau
    • Zone horaire régionale
    • Définir la date et l’heure
    • Anti-Passback : local, annulation et global.
    • Contrainte
    • LED inversée
    • Lecture continue
    • Entrées de sorties libres
  4. Cliquez sur Enregistrer.
  • Pour configurer les paramètres de porte dans NetAXS-123, référez-vous à Configurer la porte dans NetAXS123
  • Pour configurer les paramètres d'entrée de NetAXS-123, référez-vous à Configurer les entrées pour NetAXS-123
  • Pour configurer les paramètres de sortie pour NetAXS-123, référez-vous à Configurer les sorties pour NetAXS-123
  • Pour configurer les paramètres du lecteur dans NetAXS-123, référez-vous à Configurer le lecteur dans NetAXS-123
