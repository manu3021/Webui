Créer un jour férié

Le terme Jour férié est un ensemble de définitions. Par exemple, vous pouvez regrouper des jours fériés comme Noël, Thanksgiving et le Jour de l'Indépendance, qui ont lieu à la même date tous les ans, dans un groupe Jours fériés. Les groupes Jours fériés sont utiles pour regrouper des services qui fermeraient et d'autres qui resteraient ouverts pendant les jours fériés.

Chaque compte a son propre ensemble de jours fériés. Toutefois, les jours fériés communs à tous les comptes peuvent être configurés à l'aide de l'option Jours fériés.

Pour créer un jour férié :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Créer et modifier. La page CRÉER ET MODIFIER apparaît.
  2. Dans le volet gauche, vous devez sélectionner un des éléments suivants pour créer un jour férié :
    • compte distributeur
    • compte client
  3. Affichez le compte et cliquez sur Programmation, Jours fériés. L'écran Paramètres apparaît.
  4. Cliquez sur Jour férié, puis sur l'option Créer un nouveau jour férié. La page CRÉER UN JOUR FÉRIÉ PERSONNALISÉ apparaît.
  5. Entrez un Nom de jour férié.
  6. Dans Jour férié simple, cliquez sur l'icône pour sélectionner une date.
  7. Sélectionnez la case à cocher Tous les ans si le jour férié doit se répéter tous les ans.
  8. Sélectionnez la catégorie Jour férié comme Jour férié 1 ou Jour férié 2 ou Jour férié 3. Les groupes de jours fériés sont regroupés dans trois grandes catégories comme Jour férié 1, Jour férié 2 et Jour férié 3. Vous pouvez utiliser ces catégories pour regrouper les jours fériés obligatoires et les jours fériés facultatifs.
  9. Cliquez sur Enregistrer pour enregistrer le jour férié.
  10. Répétez les étapes 5 à 8 pour ajouter de nouveaux jours fériés à un groupe de jours fériés.
