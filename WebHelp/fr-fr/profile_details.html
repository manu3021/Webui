Gestion d'un profil

Vous pouvez modifier les détails d'un distributeur, vos paramètres de profil etc. et les enregistrer dans le Profil.

Pour modifier un profil :

  1. dans l'onglet Profil, cliquez sur Mon profil. La page PROFIL apparaît.
  2. Dans l'onglet Détails du distributeur, vous devez :
    • Entrer les informations du compte distributeur.
    • Charger la photo de profil du distributeur.
    • Sélectionner le programme du distributeur autorisé correspondant.
  3. Cliquez sur Suivant.
    Ou
    Cliquez sur l'onglet Mon profil .
  4. Dans l'onglet Mon profil vous devez :
    • Entrer les informations du compte client.
    • Charger la photo de profil du client.
  5. Cliquez sur Enregistrer
    Le profil est modifié.

Remarque :

pour modifier l'ancien mot de passe :

  1. cliquez sur Changer le mot de passe.
  2. Entrez l'ancien mot de passe.
  3. Entrez le nouveau mot de passe.
  4. cliquez sur Changer le mot de passe.
