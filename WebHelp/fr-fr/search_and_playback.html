Chercher et lire

La fonction Clip vous permet de rechercher une vidéo enregistrée avec une caméra particulière.

Pour visionner un clip :

  1. dans l'espace tableau de bord MPC, cliquez sur l'onglet Visionneuse.
  2. Cliquez sur l'icône. Le volet Liste universelle apparaît.
  3. Dans la Liste universelle, sélectionnez une seule caméra.

    4. Remarque :

    • dans la Liste universelle, cliquez sur Rechercher un clip pour sélectionner et rechercher plusieurs caméras.
    • Pour sélectionner toutes les caméras du site, sélectionnez la case à cocher du site.
  4. Faites un glisser-déposer de la caméra vers le panneau Visionneuse pour montrer une vidéo en direct.
  5. Cliquez sur Clip en bas à gauche de l'écran. L'option de recherche instantanée d'une seule caméra s'affiche.
  6. Dans la fenêtre de recherche d'un clip, cliquez sur l'icône. Vous pouvez désormais filtrer la recherche des vidéos enregistrées, en fonction des critères de recherche comme la vidéo enregistrée aujourd'hui, hier ou recherche personnalisée.
  7. Sélectionnez un des critères de recherche suivants correspondant au filtre date/heure.
    • Aujourd'hui : vidéo enregistrée aujourd'hui dans MPC.
    • Hier : vidéo enregistrée hier dans MPC.
    • Personnalisée : vous pouvez personnaliser la recherche.
  8. Sélectionnez un des éléments suivants pour le stockage de clip et le type de clip :
    • Type de clip
      • Événement de mouvement
      • Programmation
      • Événement déclencheur
      • Déclenchement utilisateur
      • Commandé par des règles
    • Stockage de clip
      • Sur le site
      • Dans le cloud

    9. Remarque :

    le type de programmation n'est pas pris en charge par le stockage Dans le cloud.

  9. Cliquez sur Rechercher.
