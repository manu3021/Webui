Créer et activer des règles

Pour créer et activer des règles :

  1. Dans l'onglet Règles, cliquez sur Créer une nouvelle règle.
  2. Dans Nom de la règle, saisissez un nom de règle.
  3. Dans Position, sélectionnez le site requis. QUAND vous permet de sélectionner un état pour le périphérique, une programmation et des conditions de temps.
  4. Dans Sélectionner une condition, vous pouvez définir les conditions des éléments suivants :
    • Périphérique
      • Cliquez sur l'icône dans le champ Périphérique .
      • Dans SÉLECTIONNER DES PÉRIPHÉRIQUES, sélectionnez pour inclure des périphériques tels que MPCA, panneau NetAXS, caméras, porte, entrée et sortie.
      • Cliquez sur Confirmer.
      • Cliquez sur l'icône dans le champ Sélectionner le type d'événement.
      • Dans SÉLECTIONNER LE TYPE D'ÉVÉNEMENT, sélectionnez pour définir les événements.
      • Cliquez sur Confirmer.
    • Programmation
      • Cliquez sur l'icône dans le champ Programmation .
      • Dans SÉLECTIONNER LE TYPE D'ÉVÉNEMENT, sélectionnez pour définir la programmation. Pour en savoir plus sur la configuration d'une programmation, référez-vous à créer une programmation.
      • Cliquez sur Confirmer.
    • Conditions de temps
      • Dans Dans un délai de, utilisez les flèches pour entrer les détails des min et sec .

    Remarque :

    vous pouvez aussi définir de nombreuses conditions ET/OU.

  5. Dans ENSUITE, cliquez sur Sélectionner une action.
    • Si vous sélectionnez Périphérique, alors :
      • cliquez sur l'icône dans le champ Sélectionner des périphériques .
      • Dans SÉLECTIONNER DES PÉRIPHÉRIQUES, sélectionnez pour inclure des périphériques.
      • Cliquez sur Confirmer.
      • Cliquez sur l'icône dans le champ Sélectionner une action .
      • Dans SÉLECTIONNER DES PÉRIPHÉRIQUES, sélectionnez pour inclure des actions sur le périphérique concerné.
      • Cliquez sur Confirmer.
    • Si vous sélectionnez Actions, alors :
      • cliquez sur l'icône dans le champ Type d'action .
      • Dans SÉLECTIONNER DES ACTIONS, sélectionnez pour inclure :
        • E-MAIL<>ET/OU
        • NOTIFICATIONS PUSH
      • Cliquez sur Confirmer.
      • Cliquez sur l'icône dans le champ Type d'action .
        Dans SÉLECTIONNER DES E-MAILS, la liste des noms d'opérateur créés s'affiche.
      • Sélectionnez le nom d'opérateur souhaité.
      • Cliquez sur Confirmer.
  6. Cliquez sur Enregistrer la règle.

    7. Toutes les règles récemment créées sont activées par défaut.
