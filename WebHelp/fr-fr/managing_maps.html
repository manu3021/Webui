Gestion des cartes

L'onglet Cartes, selon le/les compte(s) sélectionné(s), vous permet de situer et d'afficher l'emplacement du client/site sélectionné sur la carte.

De plus, vous pouvez zoomer sur la carte pour afficher l'emplacement exact des entités sélectionnées.

Pour situer un emplacement sur la carte :

  1. Dans l'espace tableau de bord MPC, cliquez sur l'onglet Cartes. La page CARTES apparaît.
  2. En haut à gauche de la page, sélectionnez le compte.

    3. Tous les emplacements de client du compte distributeur sélectionné sont surlignés sur la carte.

Pour voir les portes et les caméras sur la carte :

  1. Dans l'espace tableau de bord MPC, cliquez sur l'onglet Cartes. La page CARTES apparaît.
  2. En haut à gauche de la page, sélectionnez le compte.

    3. Tous les emplacements de client du compte distributeur sélectionné sont surlignés sur la carte.

  3. Dans la carte, cliquez sur le nom du site pour afficher le plan des étages du site sélectionné.

    4. Dans le plan d'étages, vous pouvez voir toutes les portes et caméras configurées.

  4. Cliquez sur la porte pour voir ce qui suit :
    • État de la porte : Affiche l'état de toutes les portes associées.
    • Info/Actions : Affiche les coordonnées du site. Affiche aussi les opérations de verrouillage et déverrouillage des portes.
    • Alarmes/événements : Affiche les alarmes et les événements (les trois derniers événements de cartes) de la porte sélectionnée.
  5. Cliquez sur la caméra pour regarder l'affichage vidéo en direct.
