Programar relatórios

Pode gerar vários relatórios utilizando o MPC. Estes relatórios podem ser gerados com base em critérios de filtro. Os relatórios podem ser programados, pré-visualizados e impressos.

Como programar relatórios:

  1. Na área do Painel MPC, clique no separador RELATÓRIOS. É apresentada a página RELATÓRIOS. É apresentada uma lista de todos os relatórios com a programação e estado.
  2. Em Relatórios do cartão, clique no relatório pretendido, que é apresentado como uma hiperligação.
  3. Realize as seguintes funções para ordenar e definir filtros para o relatório.
    4. Campo Descrição
    Programar este relatório
    Formato da programação Selecione o tipo de formato para o relatório programado.
    Ligado
    Enviar para
    Utilizadores selecionados Introduza o número total de utilizadores para o envio automático de relatórios.
    Atribuir um nome ao seu relatório Introduza um nome para o seu relatório
  4. Clique em Guardar relatório.

Nota:

Também pode selecionar um Tipo de formato para o relatório.
