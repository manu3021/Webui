Atualização do firmware

Para editar os serviços do dispositivo:

  1. Na área do Painel MPC, clique no separador Criar e editar. É apresentada a página CRIAR E EDITAR.
  2. No painel esquerdo, selecione o local.
  3. Clique no separador Atualização do firmware. É apresentada uma lista com todos os dispositivos registados.
  4. A partir da lista pendente Selecionar modelo do dispositivo, selecione o dispositivo. São apresentadas todas as versões de firmware disponíveis para o tipo de dispositivo selecionado.
  5. A partir da lista pendente Atualizar para, selecione a versão relevante para o dispositivo selecionado.
  6. Na lista Nome do dispositivo, selecione o dispositivo individual.
  7. Na lista Estado da atualização, clique em Atualizar.
  8. Todas as versões de firmware do dispositivo selecionado são atualizadas para a versão pretendida.

Nota:

A opção de atualização do firmware permite selecionar dispositivos individuais para uma atualização. A atualização em massa não é suportada para a Atualização do firmware. Por exemplo, tem de selecionar individualmente o firmware de cada dispositivo e, em seguida, clicar em Atualizar.

Durante a atualização de firmware, se tiver selecionado o dispositivo HRRH8, a Honeywell recomenda selecionar uma versão de firmware posterior à versão existente.
