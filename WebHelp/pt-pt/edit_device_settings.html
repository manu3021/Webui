Editar definições do dispositivo

Para editar as definições HRRH8:

  1. Na área do Painel MPC, clique no separador Criar e editar. É apresentada a página CRIAR E EDITAR.
  2. No painel esquerdo, selecione o local e, em seguida, clique duas vezes no dispositivo relevante adicionado ao local. É apresentada a página Definições.

    3. Nota:

    Antes de modificar as definições do dispositivo, certifique-se de que regista os dispositivos na nuvem.

  3. Introduza os detalhes necessários nos seguintes campos:
    • Em CONFIGURAÇÃO DE REDE,
      • Chave exclusiva: Introduza a MAC ID do dispositivo. A MAC ID deve ser obtida a partir do painel.
      • Endereço IP
      • Máscara de subrede
      • Servidor DNS
      • Porta HTTP
      • Gateway
    • Em DEFINIÇÕES DO GRAVADOR,
      • Nome do gravador
      • Versão do hardware
      • Formato de vídeo
      • Endereço Mac
      • Versão do firmware
    • Em DEFINIÇÕES DE LIGAÇÃO MAXPRO,
      • Ativar: Para modificar o URL do servidor.
    • Em DEFINIÇÕES AVANÇADAS,
      • Restaurar palavra-passe
      • Atualizar palavra-passe
    • Para configurar as definições da câmara HRRH8, consulte Para configurar a câmara em HRRH8

Para editar as definições NetAXS-123:

  1. Na área do Painel MPC, clique no separador Criar e editar. É apresentada a página CRIAR E EDITAR.
  2. No painel esquerdo, selecione o local e, em seguida, clique duas vezes no dispositivo relevante adicionado ao local. É apresentada a página Definições.

    3. Nota:

    Antes de modificar as definições do dispositivo, certifique-se de que regista os dispositivos na nuvem.

  3. Introduza os detalhes necessários nos seguintes campos:
    • Nome do painel
    • Descrição
    • Comunicações
    • Endereço do painel
    • Firmware
    • Intervalo do ciclo E/S
    • Intervalo de verificação do ciclo
    • Número de tentativas de CMD do painel
    • Tempo de espera do CMD do painel
    • Fuso horário regional
    • Definir Data e Hora
    • Anti-Passback: Local, Forgive e Global.
    • Pânico
    • LED invertido
    • Leitura contínua
    • Entradas de saídas livres
  4. Clique em Guardar.

  • Para configurar as definições de entrada de NetAXS-123, consulte Para configurar as entradas para NetAXS-123
  • Para configurar as definições de saída de NetAXS-123, consulte Para configurar as saídas para NetAXS-123
  • Para configurar as definições do Leitor NetAXS-123, consulte Para configurar o Leitor em NetAXS-123
