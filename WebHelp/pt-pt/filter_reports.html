Filtrar relatórios

Pode gerar vários relatórios utilizando o MPC. Estes relatórios podem ser gerados com base em critérios de filtro. Os relatórios podem ser programados, pré-visualizados e impressos.

Para filtrar relatórios:

  1. Na área do Painel MPC, clique no separador Relatórios. É apresentada a página RELATÓRIOS. É apresentada uma lista de todos os relatórios com a programação e estado.
  2. Em Relatórios do cartão, clique no relatório pretendido, que é apresentado como uma hiperligação.
  3. Realize as seguintes funções para ordenar e definir filtros para o relatório.
    4. Campo Descrição
    Conta A partir da lista de contas disponíveis, selecione a conta pretendida.
    Estado do cartão A partir da lista de estados do cartão disponíveis, selecione o estado pretendido.
    Nome do titular do cartão Introduza o nome do titular do cartão que tem de ser filtrado.
    Identidades de acesso Introduza o número de identidades de acesso associadas ao titular do cartão.
    Porta/leitores Introduza o número de portas/leitores associados ao titular do cartão.
    Selecionar data
    Data de ativação
    Data de início Selecione a data para ativar o relatório.
    Data de validade Selecione uma data de validade do relatório.
    Escolher colunas
    Selecionar colunas Selecione as colunas a apresentar no relatório.
  4. Clique em Pré-visualizar.
