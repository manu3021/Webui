Criar relatórios

A opção Criar relatórios permite criar os seguintes relatórios:

  • Auditoria
  • Cartão
  • Evento

Para criar um relatório:

  1. Na área do Painel MPC, clique no separador Relatórios.
  2. Clique em Criar relatórios. É apresentada a página RELATÓRIOS.
  3. Na coluna Relatórios, selecione o relatório pretendido. Por exemplo, selecione RELATÓRIOS DE CARTÕES
  4. Execute as seguintes funções.
Campo Descrição
FILTROS
Todos Clique para selecionar todos os relatórios.
Intervalo dos números de cartões Clique para definir um intervalo para os intervalos dos números de cartões.
Conta A partir da lista de contas disponíveis, selecione a conta pretendida.
DATA DE ATIVAÇÃO
De e Até Para especificar a data de ativação, introduza o intervalo de datas de ativação do cartão nos campos De e em Até. Pode utilizar para encontrar as datas.
DATA DE VALIDADE
De e Até Para especificar a data de validade, introduza o intervalo de datas de validade do cartão nos campos De e em Até. Pode utilizar para encontrar as datas.
Fuso horário Selecione o fuso horário do local do cartão atual.
Estado do cartão A partir da lista de estados do cartão disponíveis, selecione o estado pretendido.
Nome do titular do cartão Introduza o nome do titular do cartão que tem de ser filtrado.
ESCOLHER COLUNAS
Selecionar colunas Clique no ícone para definir as colunas a apresentar a exibição do relatório.
ORDENAR
Ordenação1 A partir da lista pendente, selecione ordenar entre número de cartão e data de ativação.
Clique no ícone para definir a ordenação ascendente.
Clique no ícone para o alterar para e definir a ordenação descendente.
Ordenação2 A partir da lista pendente, selecione ordenar entre número de cartão e data de ativação.
Clique no ícone para definir a ordenação ascendente.
Clique no ícone para o alterar para e definir a ordenação descendente.
Ordenação3 A partir da lista pendente, selecione ordenar entre número de cartão e data de ativação.
Clique no ícone para definir a ordenação ascendente.
Clique no ícone para o alterar para e definir a ordenação descendente.
PROGRAMAR ESTE RELATÓRIO
Opções A partir da lista pendente, selecione as opções para o relatório.
Ligado Clique no ícone para definir a data e hora para o relatório.
Enviar para Clique no ícone para definir o destino do relatório.
Nome do relatório Introduza um nome para o relatório.
Guardar relatório Clique para guardar o relatório.
Tipo de formato Selecione a opção para definir a opção de exportação do relatório no formato Excel ou PDF.
Exportar Clique para exportar o relatório.
