Gestion de profil

Vous pouvez modifier les détails du revendeur, les paramètres de votre profil, etc. dans l’onglet Profil .

Pour modifier un profil :

  1. Dans l’onglet Profil, cliquez sur Mon profil. La page PROFIL apparaît.
  2. Dans l’onglet Détails sur le revendeur, vous devez :
    • Saisir les détails du compte de revendeur.
    • Téléverser la photo de profil du revendeur.
    • Sélectionner le programme de détaillant autorisé correspondant.
  3. Cliquez sur Suivant.
    Ou
    Cliquez sur l’onglet Mon profil .
  4. Dans l’onglet Mon profil vous devez :
    • Saisir les détails du compte de client.
    • Téléverser la photo de profil du client.
  5. Cliquez sur Enregistrer
    Le profil est alors modifié.

Remarque :

Pour changer de mot de passe :

  1. Cliquez sur Changer de mot de passe.
  2. Saisissez le Ancien mot de passe.
  3. Faites de même avec le Nouveau mot de passe.
  4. Cliquez sur Changer de mot de passe.
