Modifier les paramètres d’un site

Pour modifier les paramètres d’un site :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier). La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le site. La page Paramètres apparaît.
  3. Faites un double-clic sur le cadre en pointillés pour téléverser le plan d’étage du site sélectionné.

    4. Remarque :

    Vous ne pouvez téléverser que des plans d’étage aux formats .JPG ou .PNG.

  4. Saisissez les informations requises, puis cliquez sur Enregistrer.
