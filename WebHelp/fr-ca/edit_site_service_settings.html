Modifier les services d’un site

Pour modifier les services d’un site :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le site.
  3. Cliquez sur l’onglet Services. Les détails du plan de périphériques apparaissent.
  4. Sélectionnez Période de rétention dans les options disponibles. La période de rétention correspond au nombre de jours pendant lesquels les clips vidéo sont disponibles sur le nuage.
  5. Sélectionnez le plan désiré dans la liste des options disponibles.
