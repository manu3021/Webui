Créer un groupe

Pour créer un groupe :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le compte du client.
  3. Cliquez sur l’icône de création , puis sélectionnez Groupe. L’écran NOUVEAU GROUPE s’affiche.
  4. Saisissez les informations nécessaires concernant le compte sélectionné, puis cliquez sur Enregistrer.

Remarque :

Vous pouvez également inclure un groupe sous un groupe existant.
