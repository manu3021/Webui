Supprimer

Pour supprimer un client, un site, un groupe ou un périphérique :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet CRÉER ET MODIFIER. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le client, le site, le groupe ou le périphérique, puis cliquez sur Supprimer. Un message d’avertissement apparaît. Vous pouvez supprimer les éléments suivants :
    • Client
    • Site
    • Groupe
    • Périphérique

    3. Une fois le client, site, groupe ou périphérique sélectionné, un message de confirmation apparaît.

  3. Cliquez sur Oui.

Remarque :

Toutes les entités du nœud parent sont supprimées.
