Ajout et modification d’opérateurs

Les opérateurs sont des personnes disposant de certaines autorisations permettant de travailler avec le système MPC. Ils peuvent se connecter à MPC au moyen d’un nom d’utilisateur et d’un mot de passe. Les opérateurs sont désignés et autorisés par l’administrateur.

Pour ajouter et modifier un opérateur :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, cliquez sur le compte du client/revendeur.

Pour créer un nouvel opérateur :

  1. Cliquez sur l’onglet Opérateurs.
  2. Cliquez sur l’icône . L’écran NOUVEL OPÉRATEUR s’affiche.
  3. Cliquez sur l’option Choisir une photo... pour parcourir, sélectionner et téléverser la photo de l’opérateur.

    4. Remarque :

    Les formats d’image pris en charge sont les suivants : .JPG, .JPEG, .PNG et .GIF. L’image doit faire moins de Suivant0 Ko.

  4. Dans la liste déroulante Modèle de rôle, sélectionnez le rôle désiré pour l’attribuer à l’opérateur.
  5. Saisissez les informations requises, puis cliquez sur Enregistrer.

Pour modifier un opérateur :

  1. Sélectionnez le compte dans le panneau de gauche, puis cliquez sur l’onglet Opérateurs.
  2. Modifiez les informations requises, puis cliquez sur Enregistrer.

Remarque :

Vous ne pouvez pas modifier l’adresse électronique servant d’identifiant dans le champ Adresse électronique principale (identifiant de connexion).
