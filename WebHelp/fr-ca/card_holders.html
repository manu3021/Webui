Titulaires de cartes

Pour créer un titulaire de carte ou ajouter de nouvelles cartes :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le client, puis double-cliquez sur Personnes. La page Paramètres est affichée.
  3. Vous pouvez créer un titulaire de carte, ainsi que de nouvelles cartes.

Pour créer un titulaire de carte :

  1. Sur la page Paramètres, cliquez sur Créer un titulaire de carte. La page CRÉER UN TITULAIRE DE CARTE est affichée.
  2. Cliquez sur le cadre de photo pour sélectionner et téléverser la photo du titulaire de carte.
  3. Saisissez les informations nécessaires sous DÉTAILS SUR L’UTILISATEUR.
  4. Sous Cartes associées, vous pouvez :
    • Créer une nouvelle carte
      • OU
    • Choisir parmi les cartes disponibles

Pour créer une nouvelle carte :

  1. Sur la page CRÉER UNE NOUVELLE CARTE, saisissez les informations suivantes :
    • Dans la liste déroulante, sélectionnez Simple ou Multiples.
    • Sous Statut de la carte, sélectionnez le type désiré.
    • Sous Niveau de problème, saisissez le niveau de problème.
    • Type de carte, sélectionnez le type de
    • Sous Nombre d’utilisations,
    • Sous Numéro de carte,
    • Sous PIN
    • Sous Temporaire
    • Nombre limité d’utilisations
  2. Cliquez sur Enregistrer sur la page CRÉER UN TITULAIRE DE CARTE.

Pour choisir parmi les cartes disponibles :

  1. Cliquez sur l’option Choisir parmi les cartes disponibles. La fenêtre contextuelle CARTES DISPONIBLES apparaît.
  2. Trouvez et sélectionnez la carte désirée dans la liste des cartes disponibles, puis cliquez sur Terminé.
  3. Cliquez sur Enregistrer sur la page CRÉER UN TITULAIRE DE CARTE.
