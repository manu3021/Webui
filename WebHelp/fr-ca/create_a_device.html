Création d’un périphérique

Pour créer un périphérique :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Sélectionnez un site dans le panneau de gauche.
  3. Cliquez sur l’icône de création , puis sur Créer>Périphérique. L’écran NOUVEAU PÉRIPHÉRIQUE s’affiche.

    4. Remarque :

    Vous ne pouvez créer de périphérique que sous un site.

  4. Sélectionnez Types de périphérique dans la liste déroulante. Vous avez le choix entre HRRH8 ou Panneau NS123.
  5. Saisissez la Clé unique (identifiant MAC) pour le périphérique sélectionné.
  6. Cliquez sur Enregistrer.

Vous pouvez modifier les paramètres du périphérique.

  • Pour modifier les paramètres du périphérique, cliquez sur Modifier les paramètres du périphérique.
