Rapports

Vous pouvez générer différents rapports avec MPC. Ces rapports peuvent être générés à partir des critères de filtrage. Les rapports peuvent être classés dans l’ordre croissant et décroissant, prévisualisés et imprimés. L’onglet Rapports donne accès aux actions suivantes :

  • Voir les rapports : cette fonction sous Liste des rapports affiche une liste de tous les rapports disponibles.
  • Créer des rapports: cette fonction sous Nouveau rapport permet de créer de nouveaux rapports, comme :

    • Audit
    • Carte
    • Événement
