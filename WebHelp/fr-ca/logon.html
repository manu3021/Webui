Connexion

Pour se connecter à l’interface d’utilisateur de MPC :

  1. Saisissez l’URL de l’interface d’utilisateur Maxpro Cloud dans votre navigateur. La page Connexion à MAXPRO Cloud apparaît.
  2. Vous pouvez vous connecter de deux façons :
    • En tant qu’utilisateur enregistré
    • En tant que nouvel utilisateur

Le tableau suivant affiche les procédures de connexion pour les utilisateurs nouveaux et enregistrés.

Si vous êtes un utilisateur enregistré... Si vous êtes un nouvel utilisateur...
a. Saisissez le Nom d’utilisateur et le Mot de passe

Remarque :

Le Nom d’utilisateur est l’adresse électronique d’identification fournie pendant l’inscription.

b. Cliquez sur CONNEXION.

Remarque :

Avant de vous inscrire, veillez à obtenir les informations d’inscription et d’approbation nécessaires auprès du revendeur.

a. Cliquez sur Créer un nouveau compte La page Inscription de revendeur apparaît.
b. Saisissez les informations requises, puis cliquez sur S’inscrire

Après avoir reçu les courriels d’inscription du nouveau compte et d’activation du compte, continuez l’inscription.

Remarque :

Si vous n’arrivez pas à vous connecter, cliquez sur Besoin d’aide? pour réinitialiser votre mot de passe.
