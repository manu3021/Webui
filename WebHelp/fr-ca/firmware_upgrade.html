Mise à niveau du micrologiciel

Pour modifier les services d’un périphérique :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le site.
  3. Cliquez sur l’onglet Mise à niveau du micrologiciel. La liste de tous les périphériques enregistrés apparaît.
  4. Sélectionnez le périphérique désiré dans la liste déroulante Sélectionner un modèle de périphérique. Tous les micrologiciels disponibles pour le type de périphérique sélectionné sont alors affichés.
  5. Dans la liste déroulante Mettre à niveau vers, sélectionnez la version désirée pour le périphérique sélectionné.
  6. Sélectionnez le périphérique dans la liste Nom du périphérique.
  7. Cliquez sur Upgrade (Mettre à niveau) dans la liste Statut de mise à niveau.
  8. Tout le micrologiciel du périphérique sélectionné est alors mis à niveau vers la version désirée.

Remarque :

L’option de mise à niveau du micrologiciel vous permet de sélectionner et mettre à niveau un périphérique spécifique. La mise à niveau groupée du micrologiciel n’est pas prise en charge. Vous devez par exemple sélectionner chaque micrologiciel de périphérique, puis cliquer sur Mettre à niveau.

Si vous avez sélectionné le périphérique HRRH8 pendant la mise à niveau du micrologiciel, Honeywell vous conseille de sélectionner une version de micrologiciel plus récente que la version actuelle.
