Créer un instantané

Vous pouvez prendre un instantané d’une vidéo en direct ou enregistrée pour l’utiliser comme image. Il est possible de créer un instantané à partir d’une vidéo enregistrée.

Pour créer un instantané :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Visionneuse.
  2. Vous pouvez prendre un ou plusieurs instantanés à l’aide des méthodes suivantes :
    • Pour prendre un instantané : Sélectionnez un clip, puis cliquez sur Instantané. (OU)
    • Pour prendre plusieurs instantanés : Dans l’onglet Visionneuse, cliquez sur Instantané.
  3. Cliquez sur Instantané dans le coin inférieur gauche de l’écran. La boîte de dialogue Enregistrer sous apparaît.
  4. Saisissez un Nom.
  5. Sélectionnez Emplacement (le répertoire dans lequel vous désirez enregistrer l’instantané). Vous pouvez aussi Parcourir et sélectionner un autre répertoire cible.
  6. Sélectionnez le Format requis (JPEG) pour l’instantané.
  7. Cliquez sur OK pour enregistrer l’instantané.
