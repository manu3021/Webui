Voir les rapports

L’option Voir les rapports vous permet d’afficher les rapports suivants :

  • Audit
  • Carte
  • Événement

Pour voir un rapport :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Rapports.
  2. Cliquez sur Voir les rapports. La page RAPPORTS apparaît.
  3. Dans la colonne Liste des rapports, cliquez sur le lien hypertexte du rapport désiré. Par exemple, sélectionnez RAPPORTS DE CARTE.
  4. Utilisez les fonctions suivantes.
