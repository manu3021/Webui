Figer le flux d’alarme

Pour figer un flux d’alarmes :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Alarmes.
  2. Cliquez sur la liste déroulante Actions dans le coin supérieur droit de l’écran, puis sur Figer le flux d’alarme.

Tous les nouveaux flux d’alarmes sont alors figés.
