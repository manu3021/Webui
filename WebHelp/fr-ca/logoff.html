Déconnexion

Vous pouvez vous déconnecter de MPC dans le menu Utilisateur.

Déconnexion de MPC

  1. Sur le tableau de bord de MPC, cliquez sur dans le coin supérieur droit. Les options du menu Utilisateur apparaissent.
  2. Cliquez sur Déconnexion.
