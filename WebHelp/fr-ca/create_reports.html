Créer des rapports

L’option Créer des rapports vous permet de créer les rapports suivants :

  • Audit
  • Carte
  • Événement

Pour créer un rapport :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Rapports.
  2. Cliquez sur Créer des rapports. La page RAPPORTS apparaît.
  3. Sélectionnez le rapport désiré sous Rapports. Par exemple, sélectionnez RAPPORTS DE CARTE.
  4. Utilisez les fonctions suivantes.
