Configurer les entrées sous NetAXS-123

Pour configurer les paramètres d’entrée :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le site, puis cliquez sur le périphérique désiré associé au site.

    3. Remarque :

    Avant de modifier les paramètres du périphérique, veillez à bien enregistrer le périphérique sur le nuage.

  3. Dans le panneau de gauche, cliquez sur l’icône correspondant au périphérique NetAXS-123 pour développer et afficher toutes les portes disponibles.
  4. Faites un double-clic sur la porte désirée pour développer et afficher toutes les entrées associées. La page Paramètres apparaît.
  5. Saisissez les informations nécessaires dans les champs suivants :
    6. Champ Description
    Nom de l’entrée Entrez un nom unique pour identifier les entrées de la porte.
    Intervalle de dérivation Indique la durée pendant laquelle l’entrée sera ignorée.
    Bloquer la durée de verrouillage Cette fonction vous permet de bloquer la programmation du verrouillage ou l’interaction programmée entre deux points.
    Désactiver les alarmes et événements Indique la durée pendant laquelle les alarmes et événements ne seront pas signalés.
    Durée de dérivation Indique la durée pendant laquelle les entrées seront dérivées ou désactivées.
    Durée de réponse Indique la durée pendant laquelle l’entrée doit maintenir son nouvel état avant de générer une alarme.
    Normalement Normalement fermé - indique que l’état normal de l’entrée est l’état fermé (par défaut).<> Normalement ouvert - - indique que l’état normal de l’entrée est l’état ouvert.
    Surveillance Indique que le circuit électrique de l’entrée est équipé de chemins alternatifs surveillés par des résistances.
    Reverrouillage automatique Provoque le reverrouillage automatique de la porte quand l’état de la porte est refermé après l’entrée. Le relais de sortie contrôlant la gâche de porte est mis hors tension quand l’entrée associée retourne à son état normal, au lieu de rester sous tension pendant la durée d’impulsion.
    Verrouillage Cette fonction vous permet de désactiver le verrouillage ou l’interaction programmée entre deux points. Quand cette fonction est activée, ce point ignore toutes les actions de verrouillage qui y sont appliquées.
  6. Cliquez sur Enregistrer.
