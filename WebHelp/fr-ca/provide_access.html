Donner accès

Pour donner un accès aux titulaires de carte :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le client.
  3. Dans le nœud Personnes, cliquez sur le lien Octroyer l’accès. La page Paramètres apparaît.
  4. Dans l’onglet Edit (Modifier), vous pouvez configurer les paramètres suivants :
    • PERSONA D’ACCÈS : dans la liste déroulante, faites votre sélection parmi les personas d’accès par défaut et configurées.
    • PORTES : sélectionnez les portes pour lesquelles vous souhaitez définir les droits d’accès.
    • PROGRAMMATIONS : sélectionnez les programmations désirées pour exécuter automatiquement les tâches à un moment donné.
  5. Cliquez sur Suivant en bas de page ou sur l’onglet Terminer.
  6. Indiquez la Date d’activation et la Date d’expiration de la persona d’accès.
  7. Donnez un nom à la persona d’accès, puis cliquez sur Enregistrer la persona d’accès ().
  8. Cliquez sur Terminer l’octroi d’accès dans le coin supérieur gauche de la page pour revenir à la page principale.
