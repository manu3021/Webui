Surveillance et affichage des alarmes

Les alarmes informent les opérateurs des actions imprévues ayant lieu. Les alarmes nécessitent une action supplémentaire de la part de l’opérateur.**

Les alarmes de faible gravité sont appelées événements. Les événements sont envoyés sous la forme de notifications.**

L’onglet Alarmes présente les éléments suivants :

  • Alarmes déclenchées
  • Événements

Les coins supérieur gauche et supérieur droit de l’onglet Alarmes affichent les options suivantes :

Numéro Champ Description
1 Tout Affiche la liste de toutes les alarmes. Vous pouvez filtrer cette liste à l’aide des critères suivants : a. Acquittées<> b. Non acquittées<> c. Événements
2 Recherche Les alarmes informent les opérateurs des événements ayant lieu. La fonction de recherche vous permet de faire des recherches dans la liste d’alarmes à l’aide des critères suivants :<> a. Entités : Caméras, Portes et Règles<> b. Moment : Aujourd’hui, Hier, 7 derniers jours, 30 derniers jours, De et À.<> c. Événement : Vidéo et Accès.
3 Actions Ce menu vous permet de réaliser les opérations suivantes pour toutes les alarmes apparaissant dans la fenêtre Alarmes. <> a. Acquitter : acquitte l’alarme sélectionnée.<> b. Effacer : efface automatiquement les alarmes lors de leur acquittement.<> c. Figer le flux d’alarme: interrompt la réception d’alarmes nouvelles ou non acquittées dans la fenêtre Alarme.<> Lorsqu’une alarme est déclenchée, elle est considérée comme non acquittée.
4 Vues Vous permet de sélectionner les options devant apparaître dans l’affichage en colonnes.
5 Acquitter Vous permet d’acquitter une alarme pour indiquer que les mesures nécessaires sont en cours d’exécution.

Lorsqu’une alarme est déclenchée, elle est considérée comme non acquittée. Vous pouvez afficher la liste de toutes les alarmes non acquittées et les actions associées dans un tableau dans l’onglet Alarmes. Les informations suivantes sont affichées pour chaque alarme non acquittée :

Champ Description
Intitulé de l’alarme Le nom de l’événement ayant déclenché l’alarme. Mouvement détecté par la caméra, par exemple. Le nom de l’attribut de l’événement. Seul l’attribut d’événement clé est affiché dans cette colonne.<>Le nom du périphérique, comme une caméra, associé à l’événement. Il peut s’agir par exemple du nom de la caméra ayant détecté un mouvement.
Type Vous pouvez faire glisser et déposer une source vidéo, comme une caméra. Vous pouvez également faire un double-clic sur le nom de la source vidéo pour la sélectionner.
Gravité Indique la gravité de l’alarme sous le type correspondant :<> a. Majeure<> b. Critique
Heure L’heure de déclenchement de l’alarme.
Date La date de déclenchement de l’alarme.
Lieu Le nom de l’emplacement du périphérique.
Action et instructions Permet d’acquitter et d’interagir avec l’alarme déclenchée.
Plus Affiche toutes les portes et tous les événements de carte associés.
