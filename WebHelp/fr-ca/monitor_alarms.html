Gestion des alarmes

La figure suivante illustre l’onglet Alarmes.

Les composants suivants sont affichés à l’écran.

Numéro Champ Description
1 Recherche Les alarmes informent les opérateurs des événements ayant lieu. La fonction de recherche vous permet de faire des recherches dans la liste d’alarmes à l’aide des critères suivants :<> a. Caméras<> b. Portes<> c. Règles<> d. Heure
2 Actions Ce menu vous permet de réaliser les opérations suivantes pour toutes les alarmes apparaissant dans la fenêtre Alarmes. <> a. Acquitter : acquitte l’alarme sélectionnée.<> b. Effacer : efface automatiquement les alarmes lors de leur acquittement.<> c. Figer le flux d’alarme : interrompt la réception d’alarmes nouvelles ou non acquittées dans la fenêtre Alarme.<> Lorsqu’une alarme est déclenchée, elle est considérée comme non acquittée.
3 Vues Vous permet de sélectionner les options devant apparaître dans l’affichage en colonnes.
4 Acquitter Vous permet d’acquitter une alarme pour indiquer que les mesures nécessaires sont en cours d’exécution.
