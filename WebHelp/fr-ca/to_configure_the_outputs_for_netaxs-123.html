Configurer les sorties sous NetAXS-123

Pour configurer les paramètres de sortie :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le site, puis cliquez sur le périphérique désiré associé au site.

    3. Remarque :

    Avant de modifier les paramètres du périphérique, veillez à bien enregistrer le périphérique sur le nuage.

  3. Dans le panneau de gauche, cliquez sur l’icône correspondant au périphérique NetAXS-123 pour développer et afficher toutes les portes disponibles.
  4. Faites un double-clic sur la porte désirée pour développer et afficher toutes les sorties associées. La page Paramètres apparaît.
  5. Saisissez les informations nécessaires dans les champs suivants :
    6. Champ Description
    Nom de la sortie Entrez un nom unique pour identifier les sorties de la porte.
    Fuseau horaire actif Permet d’indiquer deux programmations :
    • Sous tension - définit la période pendant laquelle la sortie est automatiquement sous tension.
    • Désactiver le verrouillage - définit la période pendant laquelle le verrouillage, une interaction programmée entre les entrées, sorties et groupes sélectionnés, sera désactivé. Pendant l’intervalle sélectionné, ce point ignore toutes les actions de verrouillage qui y sont appliquées, ce qui l’empêche concrètement de devenir un composant réactif durant cet intervalle.
    Le point réagit normalement à tous les verrouillages en dehors de cet intervalle.
    Le verrouillage/basculement en dehors de l’intervalle désactive la durée d’impulsion Permet d’alterner l’état des sorties entre sous tension et hors tension à chaque activation (utilisation de code, verrouillage ou impulsion manuelle).
    Bloquer la durée de verrouillage Cette fonction vous permet de bloquer la programmation du verrouillage ou l’interaction programmée entre deux points.
    Durée d’impulsion Indique la durée pendant laquelle le périphérique maintiendra son statut anormal. Ce paramètre contrôle par exemple la durée pendant laquelle un signal sonore peut retentir ou une gâche de porte peut rester désactivée.
    Verrouillage Cette fonction vous permet de désactiver le verrouillage ou l’interaction programmée entre deux points.
  6. Cliquez sur Enregistrer.
