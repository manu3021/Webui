Modification des paramètres d’un compte

L’onglet Paramètres vous permet d’ajouter ou de modifier les paramètres d’un compte.

Pour modifier les paramètres d’un compte :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, cliquez sur le compte du client/revendeur. Sélectionnez le compte à modifier.
  3. Cliquez sur l’onglet Paramètres.
  4. Modifiez les paramètres désirés, puis cliquez sur Enregistrer les paramètres.
