Recherche et lecture

La fonctionnalité Clip vous permet de rechercher une vidéo enregistrée pour une caméra en particulier.

Pour afficher un clip :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Visionneuse.
  2. Cliquez sur l’icône . Le panneau Liste universelle apparaît.
  3. Sélectionnez une caméra dans la Liste universelle.

    4. Remarque :

    • Dans la Liste universelle, cliquez sur Recherche de clip pour sélectionner et rechercher plusieurs caméras.
    • Pour sélectionner toutes les caméras associées à un site, cochez la case du site.
  4. Faites glisser et déposez la caméra dans le panneau de la visionneuse pour afficher la vidéo en direct.
  5. Cliquez sur Clip dans le coin inférieur gauche de l’écran. L’option de recherche instantanée d’une caméra unique apparaît.
  6. Dans la fenêtre de recherche de clip, cliquez sur l’icône . Vous pouvez maintenant filtrer la recherche pour les vidéos enregistrées, en fonction de critères d’enregistrements de vidéo comme Aujourd’hui, Hier et Personnalisé.
  7. Sélectionnez un des critères de recherche suivants correspondant au filtre de date et d’heure.
    • Aujourd’hui : les vidéos enregistrées dans MPC aujourd’hui.
    • Hier : les vidéos enregistrées dans MPC hier.
    • Personnalisé : vous pouvez personnaliser la recherche.
  8. Sélectionnez une des options suivantes pour le stockage et le type de clip :
    • Type de clip
      • Événement de mouvement
      • Programmation
      • Événement déclencheur
      • Déclencheur de l’utilisateur
      • Guidé par une règle
    • Stockage de clip
      • Sur site
      • Sur le nuage

    9. Remarque :

    Le type de programmation n’est pas pris en charge par le stockage Sur le nuage.

  9. Cliquez sur Recherche.
