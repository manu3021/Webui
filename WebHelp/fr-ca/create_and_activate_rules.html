Créer et activer des règles

Pour créer et activer des règles :

  1. Dans l’onglet Règles, cliquez sur Créer une nouvelle règle.
  2. Sous Nom de la règle, saisissez un nom pour la règle.
  3. Sous Position, sélectionnez le site requis. QUAND vous permet de sélectionner une condition de périphérique, de programmation et d’heure.
  4. Sous Sélectionner une condition,vous pouvez définir des conditions pour les paramètres suivants :
    • Périphérique
      • Cliquez sur l’icône dans le champ Périphérique .
      • Sous SÉLECTIONNER DES PÉRIPHÉRIQUES, sélectionnez les périphériques à ajouter, par exemple un MPCA, un panneau NetAXS, des caméras, une porte, une entrée ou une sortie.
      • Cliquez sur Confirmer.
      • Cliquez sur l’icône dans le champ Sélectionner un type d’événement.
      • Sous SÉLECTIONNER UN TYPE D’ÉVÉNEMENT, sélectionnez les événements à définir.
      • Cliquez sur Confirmer.
    • Programmation
      • Cliquez sur l’icône dans le champ Programmation .
      • Sous SÉLECTIONNER UN TYPE D’ÉVÉNEMENT, sélectionnez celui à programmer. Pour plus d’informations sur la configuration d’une programmation, consultez Créer une programmation.
      • Cliquez sur Confirmer.
    • Conditions temporelles
      • Sous D’ici, utilisez les flèches pour saisir les valeurs Min et Sec .

    Remarque :

    Vous pouvez également configurer plusieurs conditions ET/OU.

  5. Sous ALORS, cliquez sur Sélectionner une action.
    • Si vous sélectionnez Périphérique, alors :
      • Cliquez sur l’icône dans le champ Sélectionner des périphériques .
      • Sous SÉLECTIONNER DES PÉRIPHÉRIQUES, sélectionnez les périphériques à inclure.
      • Cliquez sur Confirmer.
      • Cliquez sur l’icône dans le champ Sélectionner une action .
      • Sous SÉLECTIONNER UNE ACTION, sélectionnez les actions à inclure pour le périphérique concerné.
      • Cliquez sur Confirmer.
    • Si vous sélectionnez Actions, alors :
      • Cliquez sur l’icône dans le champ Type d’action .
      • Sous SÉLECTIONNER DES ACTIONS, sélectionnez les éléments suivants pour les inclure :
        • COURRIEL<>ET/OU
        • NOTIFICATIONS PUSH
      • Cliquez sur Confirmer.
      • Cliquez sur l’icône dans le champ Type d’action .
        Sous SÉLECTIONNER DES COURRIELS, la liste des noms d’opérateurs créés est affichée.
      • Sélectionnez le nom d’opérateur requis.
      • Cliquez sur Confirmer.
  6. Cliquez sur Enregistrer la règle.

    7. Toutes les règles nouvellement créées sont activées par défaut.
