Modifier les paramètres du périphérique

Pour modifier les paramètres du HRRH8 :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier). La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le site, puis double-cliquez sur le périphérique désiré associé au site. La page Paramètres apparaît.

    3. Remarque :

    Avant de modifier les paramètres du périphérique, veillez à bien enregistrer le périphérique sur le nuage.

  3. Saisissez les informations nécessaires dans les champs suivants :
    • Sous CONFIGURATION DU RÉSEAU,
      • Clé unique : saisissez l’identifiant MAC du périphérique. Il doit être obtenu depuis le panneau.
      • Adresse IP
      • Masque de sous-réseau
      • Serveur DNS
      • Port HTTP
      • Passerelle
    • Sous PARAMÈTRES DE L’ENREGISTREUR,
      • Nom de l’enregistreur
      • Version du matériel
      • Format vidéo
      • Adresse MAC
      • Version du micrologiciel
    • Sous PARAMÈTRES DE MAXPRO CONNECT,
      • Activer : pour modifier l’URL du serveur.
    • Sous PARAMÈTRES AVANCÉS,
      • Restaurer le mot de passe
      • Mettre à jour le mot de passe
    • Pour configurer les paramètres de caméra de HRRH8, consultez Pour configurer la caméra sous HRRH8

Pour modifier les paramètres du NetAXS-123 :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le site, puis double-cliquez sur le périphérique désiré associé au site. La page Paramètres apparaît.

    3. Remarque :

    Avant de modifier les paramètres du périphérique, veillez à bien enregistrer le périphérique sur le nuage.

  3. Saisissez les informations nécessaires dans les champs suivants :
    • Nom du panneau
    • Description
    • Communication
    • Adresse de panneau
    • Micrologiciel
    • Intervalle d’interrogation E/S
    • Intervalle de vérification en boucle
    • Nombre de tentatives de la CMD de panneau
    • Délai d’expiration de la CMD de panneau
    • Fuseau horaire régional
    • Définir la date et l’heure
    • Antiretour : Local, Pardonner et Global.
    • Contrainte
    • Inverser la DEL
    • Lecture continue
    • Entrées de sortie libre
  4. Cliquez sur Enregistrer.
  • Pour configurer les paramètres de porte de NetAXS-123, consultez Configurer la porte sous NetAXS123
  • Pour configurer les paramètres d’entrée de NetAXS-123, consultez Configurer les entrées sous NetAXS-123
  • Pour configurer les paramètres de sortie de NetAXS-123, consultez Configurer les sorties sous NetAXS-123
  • Pour configurer les paramètres de lecteur de NetAXS-123, consultez Configurer le lecteur sous NetAXS-123
