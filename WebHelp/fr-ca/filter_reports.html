Filtrer les rapports

Vous pouvez générer différents rapports avec MPC. Ces rapports peuvent être générés à partir des critères de filtrage. Il est possible de programmer, de prévisualiser et d’imprimer les rapports.

Pour filtrer les rapports :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Rapports. La page RAPPORTS apparaît. La liste de tous les rapports, avec leurs programmations et statuts respectifs, apparaît.
  2. Dans la colonne Rapports de carte, cliquez sur le rapport désiré. Il est affiché sous la forme d’un lien hypertexte.
  3. Utilisez les fonctions suivantes pour trier et filtrer les rapports.
    4. Champ Description
    Compte Sélectionnez le compte désiré dans la liste des comptes disponibles.
    Statut de la carte Sélectionnez le statut désiré dans la liste des statuts de carte disponibles.
    Nom du titulaire de carte Saisissez le nom de titulaire de carte devant être filtré.
    Personas d’accès Saisissez le nombre de personas d’accès associées au titulaire de carte.
    Porte/lecteurs Saisissez le nombre de portes/lecteurs associés au titulaire de carte.
    Sélectionner une date
    Date d’activation
    Date de début Sélectionnez la date à laquelle le rapport sera activé.
    Date d’expiration Sélectionnez la date à laquelle le rapport arrivera à expiration.
    Choix de colonnes
    Sélectionner des colonnes Sélectionnez les colonnes qui apparaîtront dans le rapport affiché.
  4. Cliquez sur Aperçu.
