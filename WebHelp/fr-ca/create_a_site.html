Créer un site

Les comptes sont définis en ajoutant un site.

Pour créer un site :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Dans le panneau de gauche, sélectionnez le compte du client.
  3. Cliquez sur l’icône de création , puis sur Site. L’écran NOUVEAU SITE s’affiche.
  4. Saisissez les informations nécessaires concernant le compte sélectionné, puis cliquez sur Enregistrer.

Remarque :

Vous pouvez cliquer sur Parcourir et téléverser un plan d’étage pour trouver et téléverser le plan d’étage du site. Les formats de fichier pris en charge sont JPEG et PNG.

Pour plus d’informations sur comment configurer le plan d’étage, consultez Configuration du plan d’étage.

Vous pouvez modifier le site et les paramètres de service du site.

  • Pour modifier les paramètres du site, cliquez sur Modifier les paramètres du site.
  • Pour modifier les paramètres de service du site, cliquez sur Modifier les paramètres de service du site.

Vous pouvez mettre le micrologiciel à niveau.

  • Pour cela, cliquez sur Mise à niveau du micrologiciel.
