Trier les rapports

Vous pouvez générer différents rapports avec MPC. Ces rapports peuvent être générés à partir des critères de filtrage. Il est possible de programmer, de prévisualiser et d’imprimer les rapports.

Pour trier les rapports :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet RAPPORTS. La page RAPPORTS apparaît. La liste de tous les rapports, avec leurs programmations et statuts respectifs, apparaît.
  2. Dans la colonne Rapports de carte, cliquez sur le rapport désiré. Il est affiché sous la forme d’un lien hypertexte.
  3. Utilisez les fonctions suivantes pour créer des filtres pour les rapports.
    4. Champ Description
    Trier
    Ordre de tri 1 Définissez le premier type de tri avec lequel filtrer les rapports.
    Ordre de tri 2 Définissez le deuxième type de tri avec lequel filtrer les rapports.
    Ordre de tri 3 Définissez le troisième type de tri avec lequel filtrer les rapports.
  4. Cliquez sur Aperçu.
