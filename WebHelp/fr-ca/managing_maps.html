Gestion des cartes

L’onglet Cartes, basé sur le ou les comptes sélectionnés, vous permet de localiser et d’afficher l’emplacement du client/site sélectionné sur la carte.

Il est aussi possible de zoomer sur la carte pour afficher la position exacte des entités sélectionnées.

Pour afficher un emplacement sur la carte :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Cartes. La page CARTES apparaît.
  2. Sélectionnez le compte dans le coin supérieur gauche de la page.

    3. Tous les emplacements de client pour le compte de revendeur sélectionné sont mis en évidence sur la carte.

Pour afficher les portes et caméras sur la carte :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Cartes. La page CARTES apparaît.
  2. Sélectionnez le compte dans le coin supérieur gauche de la page.

    3. Tous les emplacements de client pour le compte de revendeur sélectionné sont mis en évidence sur la carte.

  3. Sur la carte, cliquez sur le nom d’un site pour en afficher le plan d’étage.

    4. Toutes les portes et caméras configurées sont affichées sur le plan d’étage.

  4. Cliquez sur une porte pour afficher les éléments suivants :
    • Door Status (Statut de porte) : affiche le statut de toutes les portes associées.
    • Info/Actions : affiche les coordonnées du site. Affiche également les actions de verrouillage et déverrouillage de porte.
    • Alarmes/événements : affiche les alarmes et événements (les trois derniers événements de carte) pour la porte sélectionnée.
  5. Cliquez sur la caméra pour ouvrir l’affichage vidéo en direct.
