Configuration du plan d’étage

L’onglet Créer et modifier vous permet de configurer le plan d’étage.

Pour configurer le plan d’étage :

  1. Sur le tableau de bord de MPC, cliquez sur l’onglet Créer et modifier. La page CRÉER ET MODIFIER s’affiche.
  2. Sélectionnez un site dans le panneau de gauche. L’onglet Cartes affiche l’emplacement du site sélectionné.
  3. Cliquez sur l’icône du site sur la carte pour afficher le plan d’étage du site.
  4. Toutes les portes et les caméras sont présentées dans le panneau de gauche. Pour déplacer les caméras et portes désirées sur le plan d’étage, faites un glisser-déposer.
  5. Cliquez sur Enregistrer.
