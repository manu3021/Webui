Create a Schedule

A schedule is a planned task that must be performed at the defined time periods.

Every account in MPC has a unique schedule.

You can schedule tasks to run automatically at a defined time. To schedule a task for a specific account, select the account before creating the schedule.

To create a time schedule:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, you must select any one of the following to create a schedule:
    • Dealer account
    • Customer account
  3. Expand the account and then click Schedule, Holidays. The Settings screen appears.
  4. Click Schedules and then click the Create a New Schedule option. The TIME SCHEDULE page appears.
  5. Type a Name of the Schedule and a brief Description.
  6. Click any of the weekdays and drag the mouse pointer to reach the end time of the time slot.

    7. Tip:

    It is sufficient to define the time slot for Monday, so that you can copy the time slot for the rest of the weekdays using the Copy time schedule option.

  7. Set the time slots for holidays in H1, H2, and H3.
  8. Click Save.

To edit a time schedule:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, you must select any one of the following to create a schedule:
    • Dealer account
    • Customer account
  3. Expand the account and then click Schedule, Holidays. The Settings screen appears.
  4. Click Schedules and then select schedule that must be modified.
  5. Click Edit. The TIME SCHEDULE page appears.
  6. Click any of the weekdays time schedule bar and drag the mouse pointer to reach the end time of the time slot.
  7. Click Save.

To delete a time schedule:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, you must select any one of the following to create a schedule:
    • Dealer account
    • Customer account
  3. Expand the account and then click Schedule, Holidays. The Settings screen appears.
  4. Click Schedules and then select schedule that must be deleted.
  5. Click Delete. A confirmation message appears.
  6. Click Yes.
