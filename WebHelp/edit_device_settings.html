Edit Device Settings

To edit HRRH8 settings:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, select the site and then double-click the relevant device added to the site. The Settings page appears.

    3. Note:

    Before you modify the device settings, ensure to register the devices on cloud.

  3. Enter the required details in the following fields:
    • Under NETWORK CONFIGURATION,
      • Unique Key: Type the MACID of the device. The MACID must be obtained from the panel.
      • IP Address
      • Subnet Mask
      • DNS Server
      • HTTP Port
      • Gateway
    • Under RECORDER SETTINGS,
      • Name of the Recorder
      • Hardware Version
      • Video Format
      • Mac Address
      • Version of the Firmware
    • Under MAXPRO CONNECT SETTINGS,
      • Enable: To modify the server URL.
    • Under ADVANCED SETTINGS,
      • Restore Password
      • Update Password
    • To configure the HRRH8 camera settings, refer To Configure the Camera under HRRH8

To edit NetAXS-123 settings:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, select the site and then double-click the relevant device added to the site. The Settings page appears.

    3. Note:

    Before you modify the device settings, ensure to register the devices on cloud.

  3. Enter the required details in the following fields:
    • Name of the Panel
    • Description
    • Communication
    • Panel Address
    • Firmware
    • I/O Poll Interval
    • Loop Verification Interval
    • Panel CMD Retry Count
    • Panel CMD timeout
    • Regional Time Zone
    • Set Date Time
    • Anti-Passback: Local, Forgive, and Global.
    • Duress
    • Reverse LED
    • Continuos Read
    • Free Egress Inputs
  4. Click Save.
  • To configure the NetAXS-123 Door settings, refer To Configure the Door under NetAXS123
  • To configure the NetAXS-123 Input settings, refer To Configure the Inputs for NetAXS-123
  • To configure the NetAXS-123 Output settings, refer To Configure the Outputs for NetAXS-123
  • To configure the NetAXS-123 Reader settings, refer To Configure the Reader under NetAXS-123
