Create and Activate Rules

To create and activate rules:

  1. In the Rules tab, click Create new rule.
  2. Under Rule Name, type a name for the rule.
  3. Under Position, select the required site. WHEN enables you to select a condition for the device, schedule, and time conditions.
  4. Under Select a condition, you can set conditions for the following:
    • Device
      • Click the icon in Device field.
      • Under SELECT DEVICES, select to include devices such as, MPCA, NetAXS panel, cameras, door, input and output.
      • Click Confirm.
      • Click the icon in the Select event type field.
      • Under SELECT EVENT TYPE, select to set the events.
      • Click Confirm.
    • Schedule
      • Click the icon in the Schedule field.
      • Under SELECT EVENT TYPE, select to set the schedule. For more information on configuring a schedule, refer to create a schedule.
      • Click Confirm.
    • Time Conditions
      • Under Within, use the arrows to enter the Min and Sec details.

    Note:

    You can also set multiple AND/OR conditions.

  5. Under THEN, click Select an action.
    • If you select Device, then:
      • Click the icon in the Select Devices field.
      • Under SELECT DEVICES, select to include devices.
      • Click Confirm.
      • Click the icon in the Select an action field.
      • Under SELECT DEVICES, select to include actions to the relevant device.
      • Click Confirm.
    • If you select Actions, then:
      • Click the icon in the Action type field.
      • Under SELECT ACTIONS, select to include:
        • EMAIL
          AND/OR
        • PUSH NOTIFICATIONS
      • Click Confirm.
      • Click the icon in the Action Type field.
        Under SELECT EMAILS, the list of created operator names are displayed.
      • Select the required operator name.
      • Click Confirm.
  6. Click Save Rule.

    7. By default, all the newly created rules are active.
