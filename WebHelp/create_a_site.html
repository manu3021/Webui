Create a Site

Accounts are defined by adding a site.

To create a site:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, select the customer account.
  3. Click the create icon and then click Site. The NEW SITE screen appears.
  4. Enter the required details of the site for the selected account and then click Save.

Note:

You can click Browse and Upload Floor Plan to browse and upload the floor plan to the site. The supported file formats are JPEG and PNG.

For more details on how to configure the floor plan, refer to Floor Plan Configuration.

You can edit the site and site service settings.

  • To edit the site settings, click Edit Site Settings.
  • To edit the site service settings, click Edit Site Service Settings

You can upgrade the firmware.

  • To upgrade the firmware, click Firmware Upgrade.
