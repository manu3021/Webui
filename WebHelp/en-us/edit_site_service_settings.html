Edit Site Services

To edit site services:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, select the site.
  3. Click the Services tab. The details of the device plan appears.
  4. Select the Retention Period from the available option. Retention period is the number of days the video clips are available on cloud.
  5. Select the required plan from the list of available options.
