Create a Group

To create a group:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, select the customer account.
  3. Click the create icon and then select Group. The NEW GROUP screen appears.
  4. Enter the required details of the group for the selected account and then click Save.

Note:

You can also include a group under an existing group.
