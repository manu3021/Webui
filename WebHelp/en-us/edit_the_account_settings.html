Edit the Account Settings

The Settings tab enables you to add or modify the account settings.

To edit the account settings:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, click the customer/dealer account. Select the account to be edited.
  3. Click the Settings tab.
  4. Modify the required settings and click Save Settings.
