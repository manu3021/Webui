Create a Holiday

Holiday is a set of holiday definitions. For example, you can group holidays like Christmas, Thanksgiving, and Independence Day which occur on the same date every year into a Holiday group. Holiday groups are useful for grouping departments which would close and others which would remain open, on holidays.

Each account has its own set of holidays. However, holidays common to accounts can be configured using the Holidays option.

To create a holiday:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, you must select any one of the following to create a holiday:
    • Dealer account
    • Customer account
  3. Expand the account and then click Schedule, Holidays. The Settings screen appears.
  4. Click Holiday and then click the Create New Holiday option. The CREATE CUSTOM HOLIDAY page appears.
  5. Type a Name of the Holiday.
  6. Under Single Holiday, click the icon to select a date.
  7. Select the Every Year check box, if the holiday must recur every year.
  8. Select the holiday category as Holiday 1 or Holiday 2 or Holiday 3. The holiday groups are grouped into three major categories as Holiday 1, Holiday 2, and Holiday 3. You can use these categories to group the mandatory holidays and optional holidays.
  9. Click Save to save the holiday.
  10. Repeat steps 5 through 8 to add new holidays to a holiday group.
