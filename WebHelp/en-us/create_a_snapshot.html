Create a Snapshot

You can capture a snapshot of a live or recorded video frame and use as an image. You can create snapshots from recorded video.

To create a snapshot

  1. In the MPC Dashboard area, click the Viewer tab.
  2. You can capture one or multiple snapshot in the following methods:
    • To capture one snapshot: Select one clip and click Snapshot. (OR)
    • To capture multiple snapshots: In the Viewer tab, click Snapshot.
  3. Click Snapshot at the lower left of the screen. The Save As dialog box appears.
  4. Type a Name.
  5. Select the Location (folder in which you want to save the snapshot). You can also Browse and select a different folder location.
  6. Select the required Format (JPEG) for the snapshot.
  7. Click OK to save the snapshot.
