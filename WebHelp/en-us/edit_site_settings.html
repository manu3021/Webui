Edit Site Settings

To edit site settings:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, select the site. The Settings page appears.
  3. Double click the dotted layout to upload the floor plan of the selected site.

    4. Note:

    You can only upload floor plans which are in .JPG or .PNG format.

  4. Enter the required details and click Save.
