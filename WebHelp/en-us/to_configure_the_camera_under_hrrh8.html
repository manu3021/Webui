To Configure the Camera under HRRH8

To configure the camera settings:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, select the site and then click the relevant device added to the site.

    3. Note:

    Before you modify the device settings, ensure to register the devices on cloud.

  3. In the left pane, click the icon corresponding to the HRRH8 device to expand and view all the available cameras.
  4. Double-click the required camera. The Settings page appears.
  5. Enter the required details in the following fields:
    • Under Configure
    Field Description
    Camera State Click Enable to enable the camera state.<> Or<> Click Disable to disable the camera state.
    Camera Name Enter a name for the camera.
    Camera Type From the drop-down list, select the type of camera.
    Select Discovered IP Camera From the drop-down list, select the required IP camera.
    Camera Model From the drop-down list, select the model of the camera.
    RTSP URL Enter the URL of the Real Time Streaming Protocol (RTSP) network.
    User Name Enter the user name to access the MPC appliance URL.
    Password Enter the password to access the MPC appliance URL.
    Play Click the icon to play the clip available in the selected camera.
    Associations Click to associate the camera to any door. For example, when any event occurs at the door, such as, card found, the camera begins recording the video.
    RECORDING CLIP DURATION
    Pre Event Click the time to set how long the clip is recorded before the motion is detected by the camera.
    Onsite Post Event Click the time to set how long the clip is recorded after the motion is detected by the camera, located onsite
    Offsite Post Event Click the time to set how long the clip is recorded after the motion is detected by the camera, located offsite.
    LIVE/RECORD SETTING
    Frame Rate From the drop-down list, select the frequency (rate) at which the camera produces unique consecutive images called frames.
    Camera Resolution From the drop-down list, select the camera resolution. It refers to the total number of pixels in a digital camera image.
    Camera Quality From the drop-down list, select the quality of the camera.
    • Under Event Recording
    Field Description
    Record on From the list of available triggers, select the required trigger point at when the recording should begin.
    • Under Continuous Recording
    Field Description
    Continuous Recording Schedule From the list of available schedules, select the required period during which the continuous recording is enabled.
    • Under Motion Recording
    Field Description
    Enable Motion Recording Select to enable motion recording.
    Hold Time
    Motion Recording Schedule From the list of available schedules, select the required period during which the motion recording is enabled.
    • Click Save.
