Firmware upgrade

To edit device services:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, select the site.
  3. Click the Firmware upgrade tab. A list, consisting of all the registered device, appears.
  4. From the Select Device Model drop-down list, select the device. All the available firmware for the selected device type, appears.
  5. From the Upgrade To drop-down list, select the relevant version for the selected device.
  6. In the Device Name list, select the individual device.
  7. In the Upgrade Status list, click Upgrade.
  8. All the selected device firmware is upgraded to required version.

Note:

Firmware upgrade option enables you to select the individual devices for an upgrade. Bulk upgrade is not supported for firmware upgrade. For example, you must individually select every device firmware and then click Upgrade.

During the firmware upgrade, if you have selected HRRH8 device, Honeywell recommends you to select the firmware version later than the existing version.
