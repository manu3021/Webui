Filter Reports

You can generate a number of reports using MPC. These reports can be generated based on the filter criteria. Reports can be scheduled, previewed, and printed.

To filter reports:

  1. In the MPC Dashboard area, click the Reports tab. The REPORTS page appears. A list of all the reports, with the schedule and status, appears.
  2. Under Card Reports column, click the required report, which appears as a hyperlink.
  3. Perform the following functions to sort and set filters for the report.
    4. Field Description
    Account From the list of available accounts, select the required account.
    Card Status From the list of available card status, select the required status.
    Card Holder Name Type the name of the card holder name that must be filtered.
    Access Personas Type the number of access personas associated with the card holder.
    Door/readers Type the number of door/readers associated with the card holder.
    Select Date
    Activation Date
    Start Date Select a date to activate the report.
    Expire Date Select a date for the report expiration.
    Pick Columns
    Select Columns Select the columns to appear in the report display.
  4. Click Preview.
