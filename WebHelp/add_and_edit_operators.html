Add and Edit Operators

Operators are individuals with a set of privileges to work with the MPC system. An operator can log on to MPC using a user name and a password. Operators are assigned and provided access rights by the administrator.

To add and edit an operator:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, click the customer/dealer account.

To create a new operator:

  1. Click the Operators tab.
  2. Click the icon. The NEW OPERATOR screen appears.
  3. Click the Choose pic.. option to browse, select, and upload the operator's picture.

    4. Note:

    The supported image formats are .JPG, .JPEG, .PNG, and .GIF, with a maximum size limit of 400 KB.

  4. From the Role Template drop-down list, select to assign the required role for the operator.
  5. Enter the required details and click Save.

To edit an operator:

  1. Select the account from the left pane and then click the Operators tab.
  2. Modify the required details and click Save.

Note:

You cannot edit the e-mail ID in the Primary Email (Login ID) field.
