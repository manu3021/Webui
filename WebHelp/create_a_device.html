Create a Device

To create a device:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, you must select a site.
  3. Click the create icon and then click Create>Device. The NEW DEVICE screen appears.

    4. Note:

    You can create a device only under a site.

  4. Select the Device Types from the drop-down list. You can either select HRRH8 or NS123 Panel.
  5. Enter the Unique Key (MAC ID) for the selected device.
  6. Click Save.

You can edit the device settings.

  • To edit the device settings, click Edit Device Settings.
