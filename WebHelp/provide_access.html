Provide Access

To provide access to cardholders:

  1. In the MPC Dashboard area, click the Create & Edit tab. The CREATE AND EDIT page appears.
  2. In the left pane, select the customer.
  3. In the People node, click the Give Access link. The Settings page appears.
  4. In the Edit tab, you can set the following:
    • ACCESS PERSONA: From the drop-down list, select from the list of default and existing access personas.
    • DOORS: Select the doors for which you can set the access rights.
    • SCHEDULES: Select the required schedules to automatically run the tasks at a defined time.
  5. Click Next at the bottom of the page or click the Finish tab.
  6. Specify the Activation Date and Expiry Date of the access persona.
  7. Type a name for the access persona and then click Save Access Persona.
  8. Click Finish Giving Access at the top left of the page to return to the main page.
